Microsoft Excel

  • Make numbers negative using excel's copy and paste feature

    • Step 1: Type in number you want to multiply.
    • Step 2: Copy cell.
    • Step 3: Select cells you want to multiply.
    • Step 4: Over selected cells, right click mouse or control + mouse click.
    • Step 5: Select "Paste Special" from popup.
    • Step 6: Select "Paste Special..." from dropdown menu
    • Step 7: Select "Multiply" under the "Operations" group
    • Step 8: Click Ok
    • All numbers selected are multiplied by the value you copied.

  • Change font color for negative values in excel

    • Step 1: Select cells of interest
    • Step 2: Select Conditional Formatting in Home Ribbon, in Styles Group
    • Step 3: Select "New Rule" from drop down.
    • Step 4: Select "Classic" from Styles drown down menu.
    • Step 5: Select "Format cells that contain"
    • Step 6: Change "Specific Text" to "Cell Value"
    • Step 7: Change "Between" to "Less Than"
    • Step 8: Add "0"
    • Step 9: In "Format with" drop down menu, select "Red Text"
    • Step 10: Click OK

  • Keyboard Shortcut to Select All Continous Cells with Data

    • Tip to select all continuous cells with data.
    • Step 1: Click on the upper right cell.
    • Step 2: To select all the continuous cells with data down a column.
    • Shift + Command + Down Arrow
    • Step 3: To then select all the continuous cells with data across columns
    • Shift + Command + Right Arrow

  • Insert Pivot Table Chart

    • Insert an Excel Pivot Table Chart
    • Step 1: Select all data in worksheet
    • PC: Ctrl + a
    • Mac: Command + a
    • Step 2: Select Insert Tab in Ribbon
    • Step 3: In Charts Group, Select PivotChart
