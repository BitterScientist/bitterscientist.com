Simple Subquery¶

  • Is a query that can be evaluated independently from the outer query
  • A simple subquery is only processed once in the entire statement

Example 1: Simple Subsquery in WHERE Clause Using a Scalar Subsquery¶

  • Useful for filtering results based on information calculated separtely beforehand.

Goal: Use a subquery to return team id's that have made more home team goals than the average home team goals¶

Step 1: Evaluate the Inner Query¶

In [6]:
# This simple query is to be included in a where clause
# Note this can be processed on its own, and returns a scalar value
sql_subquery = """SELECT AVG(home_team_goal) AS "Average Goals" FROM Match;"""
In [7]:
# Run query
simple_subquery = pd.read_sql(sql_subquery, conn)
# Displaying data
simple_subquery

Step 2: Incorporate Inner Query in WHERE Clause¶

In [8]:
sql_view1 = """SELECT home_team_api_id 
            FROM Match 
            WHERE home_team_goal > (SELECT AVG(home_team_goal) FROM Match);"""

  • The outer query than uses the scalar value to filter rows that meet the condition

Example 2: Filter Data by Performing a Calculation on a Subsquery in a WHERE Clause that uses a Scalar Value¶

Goal: Retrieve teams that scored two times above the average in the 2015 soccer season¶

Step 1: Assemble Inner Query¶

SELECT AVG(home_team_goal) FROM Match)

Step 2: Incorporate Inner Query into Outer Query¶

In [10]:
sql_view2 = """SELECT home_team_api_id 
            FROM Match 
            WHERE season = '2013/2014'
                AND home_team_goal >= 
                    2 * (SELECT AVG(home_team_goal) FROM Match);"""

Explanation:

Inner Query

  • The simple subquery SELECT AVG(home_team_goal) FROM Match clause is processed first, and returns a scalar value to the outer query

Outer Query

  • Calculation step: Doubles the scalar values from inner query: 2 * (SELECT AVG(home_team_goal) FROM Match)
  • Filtering step : where two conditions are met: 1) season = 2013-2014, and the team scored twice the average in home goals WHERE season = '2013/2014' AND home_team_goal >= 2 * (SELECT AVG(home_team_goal) FROM Match)

Final Retrieved Data
All teams that have scored double the average of home runs in 2013-2014

Example 3: Subsquery in WHERE Clause using a list with the IN Clause to Filter Data¶

Goal: Retrieve all teams that are part of the British league.¶

Step 1: Assemble Inner Query¶

SELECT home_team_api_id FROM Match WHERE country_id = 1729

Step 2: Incorporate Inner Query into Outer Query¶

In [12]:
sql_view3 = """SELECT 
                    team_long_name, 
                    team_short_name AS abbr
                FROM Team 
                    WHERE team_api_id IN 
                    (SELECT home_team_api_id
                    FROM Match
                    WHERE country_id = 1729);"""
