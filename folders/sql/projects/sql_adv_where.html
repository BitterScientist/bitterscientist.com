Using the AND Operator¶

Use this to filter by more than one column.

  • AND is used to join two conditions
  • AND instructs the database to only return data that meets all specified conditions.


SQL Syntax
SELECT column(s)
FROM table WHERE column = SOMEVALUE AND column = SOMEVALUE;

Using the OR Operator¶

Opposite of AND, and is used to instruct a database to return data that matches either specified conditions.

Using Parentheses in WHERE Clauses¶

Parentheses are used to explicitly group related operators.

  • The operators used in the syntax example below is an example, the operators you use will depend on the data you are trying to retrieve.


SQL Syntax
SELECT column(s)
FROM table WHERE (columnA = SOMEVALUE OR columnB = SOMEVALUE) AND columnC = SOMEVALUE;

Using the IN Operator¶

Used to specify a range of conditions.

  • Is expressed as an enclosed list of comma-delimited values in parentheses.


SQL Syntax
SELECT column(s)
FROM table WHERE columnA IN (SOMEVALUE)
ORDER BY columm;

can be any number of valid values
