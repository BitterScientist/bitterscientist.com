Background Info

Multiple Subqueries in one query.: Is an SQL query that contains multiple subqueries in any of the following clauses:

  • SELECT Clause
  • FROM Clause
  • WHERE Clause
Why is this important?
Extracting data from sql can easily become extensive. Learning how to read and write large queries is an essential task for any analyst or data scientist.
Nested queries in several clause also enables you to perform complex queries that would be difficult to conduct otherwise.

Objective: The goal of this post is to demonstrate how to use multiple subqueries in a single statement.

Data: The data for this post was obtained from kaggle.com post European Soccer Database Accessed on 2/3/20.

Method: This post used the python libraries pandas, and SQLAlchemy were used in conjunction with jupyter notebook to run the MySQL SELECT commands. This was done for ease of displaying the results. All SQL commands are in red.
