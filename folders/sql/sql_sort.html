Background Info

The ORDER BY Statement: The ORDER BY clause explicitly sorts retrieved data by the specified column(s) in the clause.
Without the use of the ORDER BY clause, data retrieved from an initial query will be returned in the order in which it appears in the database tables. Columns can be referred to by string or column position number, and the columns that are sorted (ordered) do not have to be specified in the SELECT statement.

Objective: The goal of this post is to highlight some of the various ways we can order data from table(s) using the ORDER BY clause.
