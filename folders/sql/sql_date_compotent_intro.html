Background Info

Introduction to MySQL Dates and Time Functions: SQL has various built-in functions that are used to manipulate temporal data. This includes:

  • Functions that Extract Components of a DateTime objects.
  • Functions that manipulate DateTime objects.
  • Functions that conduct mathematical operations on DateTime objects.

Why is this important?
Manipulating temporal values in SQL is a key skill set, especially when dealing with datasets that track events across time.

Objective: The goal of this post is to demonstrate how to use extract component parts from a datetime object.

Data: The data for this post consists of artificial dates.

Method: This post used the python libraries pandas, and SQLAlchemy were used in conjunction with jupyter notebook to run the MySQL SELECT commands. This was done for ease of displaying the results. All SQL commands are in red. Please note that when using SQL statements in SQLALCHMEY, a '%%' rather than a single '%' is used. In other words, when testing these examples in MySQL, you would not need to use '%%', just '%'.
