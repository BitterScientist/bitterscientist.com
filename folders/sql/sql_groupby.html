Background Info

Grouping data: SQL has a powerful clause that allows queries to group data into subsets. This involves the use of SELECT statement clauses GROUP BY and optionally the HAVING clause. Grouping allows you to divide your data into logical sets that you can use to perform aggregated calculations on each group.

Objective: The goal of this post is to introduce how to group data, how to filter grouped data, and how to combine grouping clauses with the WHERE and logical operators.

Data: The data for this post was obtained from datacamp.com post Introduction to Portfolio Analysis in Python File: Portfolio factors.csv, accessed on 1/20/20.

Method: This post used the python libraries pandas, and SQLAlchemy were used in conjunction with jupyter notebook to run the MySQL SELECT commands. This was done for ease of displaying the results. All SQL commands are in red.
