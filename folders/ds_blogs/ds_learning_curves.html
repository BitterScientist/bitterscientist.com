Background Info

Learning curves are diagnostic plots used in machine learning to evaluate the performance of an algorithm.

    Specificially:
  • Plot informs how well a model "learns" from the training dataset as you increase the number of data.
  • How well the algorithm generalizes this learning to the validation dataset.
  • Diagnoses model underfitting and overfitting.


The goal of this post To demonstrate how to:

  • generate learning curves using python
  • diagnose model overfitting
  • strategies to adddress model overfitting
  • diagnose model underfitting
  • strategies to adddress model underfitting
