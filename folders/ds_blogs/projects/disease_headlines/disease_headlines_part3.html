DBSCAN: A Clustering Algorithm for Grouping Data Based on Spatial Density¶

  • DBSCAN clustering data composed of curving and dense shapes.
  • DBSCAN does not require an approximation of the cluster
  • DBSCAN can filter random outliers located in sparse regions of space.
  • DBSCAN cannot be reapplied to new data, a new DBCAN would need to be applied to the union of the old and new data.
  • DBSCAN does not depend on the mean

# Function that defines a NumPy-based great-circle metric
def great_circle_distance(coord1, coord2, radius=3956):
    if np.array_equal(coord1, coord2):
        return 0.0

    coord1, coord2 = np.radians(coord1), np.radians(coord2)
    delta_x, delta_y = coord2 - coord1
    haversin = m.sin(delta_x / 2) ** 2 + np.product([m.cos(coord1[0]),
                                                   m.cos(coord2[0]),
                                                   m.sin(delta_y / 2) ** 2])
    return  2 * radius * m.asin(haversin ** 0.5)

DBSCAN assumptions¶

  • min_samples: The global cities cluster around at least 3 cities
  • eps: Clusters are on average no more than 250 miles apar
