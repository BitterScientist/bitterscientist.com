The need for learning curves; a case study using non-linear data.

  • Linear models are typically considered for data that scatters in a single direction (correlate).

  • In many cases, data will be more complex than a straight line, calling for a more nuanced approach.

  • Linear models can be used to fit nonlinear data.

  • One method is to add new features that contains the powers of the original features.

  • A linear model can then be trained on this new extended data, and this technique is called Polynomial regression.

  • Using learning curves on a model can help determine if the initial machine learning algorithm used is not complex enough, or too complex

  • In this example, we will use learning curves to confirm that a polynomial regression is appropriate for data that is nonlinear.

  • This example, will also help illustrate how to evaluate the behaviors of the learning curve, and use this information to diagnose underfitting, or overfitting.

  • PolynomialFeatures(degree=d) transforms an array containing n number of features into an array containing (n+d)!/d!n! number of features.

  • Linear regression can now be performed on this new dataset.

High-Degree Polynomial Regression

In [11]:
from sklearn.preprocessing import StandardScaler
from sklearn.pipeline import Pipeline

for style, width, degree in (("g-", 1, 200), ("b--", 2, 2)):
    polybig_features = PolynomialFeatures(degree=degree, include_bias=False)
    std_scaler = StandardScaler()
    lin_reg = LinearRegression()
    polynomial_regression = Pipeline([
            ("poly_features", polybig_features),
            ("std_scaler", std_scaler),
            ("lin_reg", lin_reg),
        ])
    polynomial_regression.fit(X, y)
    y_newbig = polynomial_regression.predict(X_new)
    plt.plot(X_new, y_newbig, style, label=str(degree), linewidth=width)

plt.plot(X, y, "b.", linewidth=3)
plt.legend(loc="upper left")
plt.xlabel("$x_1$", fontsize=18)
plt.ylabel("$y$", rotation=0, fontsize=18)
plt.axis([-3, 3, 0, 10])
plt.savefig("images/poly_plots.png", bbox_inches='tight')
plt.show()

Comments.

  • In this example, the blue line is a polynomial with a degree of 2. Essentially the same plot as in the previous example.

  • The green line is a polynomial with a degree of 200.

  • As you can see in the green curve, the higher dimensionality makes the curve oscillate up and down trying to capture as best as possible, the scatter in the data.

  • This closer fit to the data translates to improved learning of the data.

  • Although learning the data is good for machine learning, it poses a problem if the model learns too much, that it becomes regid, and is unable to generalizing to new data.

  • This is where learning curves come into play, they allow us to diagnose issues of "too much learning", or not "learning enough", which would ultimately make the model unable to make good predictions.

The Goldie Locks Complex in Model Complexity.

  • As illustrated in the plot below, models that overfit the training data (high-degree polynomial regression, green curve) will not generalize well with new data.

  • Models that underfit the data (linear model, red line), will not generalize well with the training, and new data.

  • The model that generalizes best is in the Goldie Locks zone; it fits the data, but not excessively, as is displayed in can be seen with the blue line, quadratic model.

  • One way to determine is a model is to simple, or too complex, is to use learning curves.

from sklearn.pipeline import Pipeline

for style, width, degree, label in (("g-", 1, 200, "deg=1, too simple, undefit"), ("b--", 2, 2, "deg=2, generalizes"), ("r-+", 2, 1, "deg=200, too complex, overfit")):
    polybig_features = PolynomialFeatures(degree=degree, include_bias=False)
    std_scaler = StandardScaler()
    lin_reg = LinearRegression()
    polynomial_regression = Pipeline([
            ("poly_features", polybig_features),
            ("std_scaler", std_scaler),
            ("lin_reg", lin_reg),
        ])
    polynomial_regression.fit(X, y)
    y_newbig = polynomial_regression.predict(X_new)
    plt.plot(X_new, y_newbig, style, label=label, linewidth=width)
