Residuals vs Fitted

Explained
  • The residuals versus fit plot has the residuals on the y axis and the fitted values (estimated responses) on the x axis
  • The plot is used to detect non-linearity, unequal error variances, and outliers.
  • To plot the "residuals versus fits plot", you can use the seaborn plotting function:
    seaborn.residplot(x, y, data=None, lowess=False, x_partial=None, y_partial=None, order=1, robust=False, dropna=True, label=None, color=None, scatter_kws=None, line_kws=None, ax=None)
  • The x: the model fitted y value array
  • The y: the model residuals
  • Additional information on residplots: Documentation
