The Obejective (Big Picture) of Gradient Descent

  • Image you are are the top of this mountain, are blind, and want to reach the town at the base of this mountain. How would you do this?
  • One way is to use your feet, and tap in all directions around you, until you feel a direction with a slope that is descending.
  • You do this with each step, until you gradually reach the bottom.
  • Yes this will take you a very long time!
  • This is what Batch Gradient Descent seeks to do. Get you to the bottom of your cost function so that your trained model has the lowest rate of false predictions.

Explanation
  • The theta 1 is the next step you would take as you trek down the moutain to reach the town at it's base.

  • Theta zero is the current position.

  • Alpha is how big of a step you take, a small step is conservative, which means it will take you longer to reach the base, but a big step can take you in the wrong direction all together. As explained later.

  • the nabla (upside down triangle) is the gradient/slope, so this means you will go down the direction with the steepest slope.

  • Why do we subtract the size of the step and the direction with the steepest slope to the current position?


  • Answer: our current position on this mountain (or cost function) is presumably high, so we want to go down, so we subtract (or descent) down the mountain/cost function by subtracting the step we take, times the gradient from our current position

  • In machine learning terms, we use training data, data where we know the actual output, to build a model.

Cost Function Explained

  • The Cost Function/Loss Function evaluates the performance of the machine learning algorithm.

  • The Loss Function calculates the error for a single instance.

  • The Cost Function is the average of the loss function for the set of training instances.


For the formula below:
  • The N is the total number of instances in the dataset.

  • The Y^1 is the predicted value for a specific instance.

  • The Y is the actual value for a specific instance.

How the cost function is minimized


  • The Cost function is essentially of the Y = X^2, which when plotted, is a parabola (such as the gif below).

  • The point in this parabola that has the lowest cost is the minimum.

  • Gradient Descent helps fund this location.

  • If we draw tangent lines as we move throuch each point in the parabola (each y value, aka the prediction value), we can get the gradient (or steepness of the slope).

  • As the steepness approaches zero, we approach the lowest cost in our function, as illustrated in the tangent lines in the gif below.
