Confirm Project Meets Binomial Logistic Regression Assumptions

Part 1: Provide an Introduction to Logistic Regression and Model Assumptions.

  • Logistic regression is regression algorithm that can be used for classification problems.

  • Logistic regression is also known as Logit Regression

  • There are three major types of logistic regression:
    • Binomial Logistic Regression: there are only two possible outcomes (yes/no, cat 1 or cat 2).
    • Multinomial Logistic Regression: there are three or more possible outcomes without order. Example: planet, start, comet
    • Ordinal Logistic Regression: there are three or more possible outcomes with ordering. Example: movie ratings from boring, okay, good, great.
  • The focus of this post is the binomial logistic regression

  • Binomial Logistic regression is used to explain the relationship between a dependent binary variable and one or more continuous or categorical data (nominal, ordinal, interval, or ratio-level) independent variables.

  • In other words: Binomial Logistic regression is used to estimate the probability that an instance belongs to a specific class. Does the observation fall into one of two categories?

  • A decision boundary is used to set a threshold for prediction making. For binary classification, if an instance is less than 50%, it is classified in one category, and if is greater or equal to 50%, it is classified in the 2nd cagetory.

  • Decision boundaries can be linear or non-linear. Polynomial order can be increased to get a more complex decision boundary

  • Example of logistic regression problems:
    • What is the probability that email is spam? (spam/ not spam)
    • What is the probability that transaction is fraud? (yes/no)
    • What is the probability that this tissue sample is cancerous? (yes/no)

  • Instances greater than 50% are labeled positive class, "1" .

  • Instances less than 50% are labeled negative class, "0".

  • Major Assumptions

    1. Dependent variable should be dichotomous in nature (yes/no, present/absent, male/female etc). If your dependent variable is continuous, you need to carry out a multiple regression analysis instead.
      The dependent variable in this study is dichotomous (wide receiver or not wide receiver).

    2. You have one or more independent variables, which can be either continuous variables (i.e., an interval or ratio variable) or nominal variables.
      The independent variables for this study are both continuous variables (player weight in lbs and player height in inches).

    3. There should be independence of observations.
      Each instance in this dataset are individual players. Each instance is independent.

    4. The categories of the dichotomous dependent variable and all your nominal independent variables should be mutually exclusive and exhaustive.
      There are no nominal independent variables in this study.

    5. There should be a bare minimum of 15 cases per independent variable (although some recommend as high as 50 cases per independent variable).
      There are over 2652 observations, well over the minimum for this method.

    6. There needs to be a linear relationship between any of the continous independent variables and the logit transformation of the dependent variable. The Box-Tidwell (1967) procedure can be used to test for linearity.
      Box-Tindell test was conducted using SPSS as described by Laerd Statistics

    7. Your data must not show multicollinearity.
      A Variable Inflation Factor test was conducted on the weight and height features and the result suggest high multicollinearity. To resolve this issue, this binary logistic regression will only use 1 of these two features, the weight of players.

    8. There should be no significant outliers, high leverage points or highly influential points.
      Boxplot and z-score analysis were conducted on the key independent variables and no evidence to suggest outliers exist in either of these features.

Test for Multicollinearity Using Variable Inflation Factors (VIF)

In [4]:
import statsmodels.api as sm
from statsmodels.stats.outliers_influence import variance_inflation_factor

def calc_vif(X):

    # Calculating VIF
    vif = pd.DataFrame()
    vif["variables"] = X.columns
    vif["VIF"] = [variance_inflation_factor(X.values, i) for i in range(X.shape[1])]

    return(vif)

Comments on VIF:

  • The VIF is 1 divided by (1 - R^2).

  • The VIF starts at 1 and there is no upper limit.

  • If VIF is 1, then no correlations between independent variables.

  • If VIF exceeds 5, it indicates multicollinearity between the independent variables.

  • In this case, the VIF for both variables are higher than 5 and suggest multicollinearity, as a result, only one variable will serve as the independent feature. weight

from sklearn.linear_model import LogisticRegression

X = nfl_df[['Height_in', 'Weight_lbs']].values
y = (nfl_df["target"] == 1).astype(np.int)

logreg = LogisticRegression(solver="lbfgs", C=10**10, random_state=42)
logreg.fit(X, y)

x0, x1 = np.meshgrid(
        np.linspace(nfl_df.Height_in.min(), nfl_df.Height_in.max(), 500).reshape(-1, 1), #xaxis
        np.linspace(nfl_df.Weight_lbs.min(), nfl_df.Weight_lbs.max(), 200).reshape(-1, 1), #yaxis
    )
X_new = np.c_[x0.ravel(), x1.ravel()]

y_proba = logreg.predict_proba(X_new)

plt.figure(figsize=(15, 8))
plt.plot(X[y==0, 0], X[y==0, 1], "bs")
plt.plot(X[y==1, 0], X[y==1, 1], "g^")

zz = y_proba[:, 1].reshape(x0.shape)
contour = plt.contour(x0, x1, zz, cmap=plt.cm.brg)


left_right = np.array([nfl_df.Height_in.min(), nfl_df.Height_in.max()])
boundary = -(logreg.coef_[0][0] * left_right + logreg.intercept_[0]) / logreg.coef_[0][1]
