Ridge Regularization of Polynomial Linear Models using Python SciKit Learn.

First Posted: 12-28-2020
Last Updated: 12-28-2020

Background Info

Regularization

  • A model that attempts to create a polynomial function that approximates a set of data points.


The project goals

  • Provide an Introduction to Regularization.
  • Discuss Ridge Regression.
  • Demonstrate how to carry out a Ridge Regression.
  • Demonstrate how the hyperparameter alpha affects the polynomial regression curve
