Polynomial Regression in Python.

First Posted: 9-21-2020
Last Updated: 10-29-2020

Background Info

Polynomial Regression

  • A model that attempts to create a polynomial function that approximates a set of data points.


The project goals

  • Provide an Introduction to Polynomial Regression.
  • Describe when this model should be considered.
  • Demonstrate how to perform a polynomial regression using Python's Scikit-Learn library.

Resources & Citations

  • 1. ML | Normal Equation in Linear Regression.
  • Géron, Aurélien. Hands-on Machine Learning with Scikit-Learn, Keras, and Tensorflow: Concepts, Tools, and Techniques to Build Intelligent Systems. , 2019. Print.

Research Objectives

  1. Data extraction, transformation, loading, and exploring pipeline.
  2. Demonstrate how to carry out a polynomial regression.
  3. Evaluate model performance using learning curves.

Methods

Data

  1. The data for this project was obtained from Scikit-learn, and contains the Bosting Housing dataset, a standard in regression analysis.(OECD) website
  2. Belsley, Kuh & Welsch, ‘Regression diagnostics: Identifying Influential Data and Sources of Collinearity’, Wiley, 1980. 244-261.
  3. Quinlan,R. (1993). Combining Instance-Based and Model-Based Learning. In Proceedings on the Tenth International Conference of Machine Learning, 236-243, University of Massachusetts, Amherst. Morgan Kaufmann.

Analysis
 The programming language Python was used in this project. The matplotlib library was used to visualize the data. Pandas was used to wrangle the data, and the numpy, and Scikit-Learn libraries were used in the calculations and machine learning models.
