Microsoft Office Word

  • Add Auto-numbering Rows in Table Column

    • Step 1: Hover mouse cursor over column header.
    • Step 2: Mouse click when arrow appears.
    • Step 3: Select "Numbering" icon in paragraph group.
    • Step 4: Select "Decrease Indent" icon in paragraph group.
    • Row numbers auto update as rows are added or removed.

  • Use Mail Merge to Print Several Documents

    • Step 1: Draft document
    • Step 2: Use symbols to demarcate fields to merge (optional).
    • Step 3: In Mailings Ribbon, "Select Recipients" in "Start Mail Merge Group".
    • Step 4: Select User Existing List (you must already have a csv file with merge data).
    • Step 5: Insert Merge Fields by selecting "Merge Fields" in "Write and Insert Fields" group
    • Step 6: Complete merge by selecting "Finish & Merge" in "Finish Group", and then select "Print Individual Documents"
