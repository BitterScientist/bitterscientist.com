How to Clean Column Headers Using Pandas and Regex.

Background Info

Pandas and Regex: Pandas combines the power of regular expression to manipulate strings. Regular expressions is a method to find patterns in string sequences.

Why is this important?
Exporting data from various sources, such as adobe fillable forms or even qualtrics, will result in column headers containing metadata information about the fields the data was exported. Knowing how to manipulate these strings to make the column headers have meaningful titles is an essential part of the data wrangling process.

The Objective
The objective of this post is to demonstrate how to manipulate column headers strings, so that the column titles do not contain metadata information imported from their data source.
This will only work if each column header has the same pattern.
This is recommended if there are many columns, and having to manually manipulate each column would be time consuming.

Data source: The column headers were obtained by exporting data from an adobe pdf fillable form.
