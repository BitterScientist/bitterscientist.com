Background Info

Python Dictionaries: are Python's associative arrays or hash tables that consist of a collection of key-value pairs. Each key is connected to a value, and can be used to connect pieces of related information in an efficient manner.
Python dictionaries are created with an empty set of braces, and each key-value are separated by a colon, while each pair of key-values are separated by a comma.
Because dictionaries resemble lists to some degree, below is a comparison of these two data structures.
Comparison of Dictionaries and Lists

  • Dictionary values can be accessed by means of integers, strings, or other python objects. Lists are accessed by means of integers (indices).
  • Both dictionaries and lists can store any python object
  • The position of values stored in dictionaries are NOT implicitly ordered, whereas the values in a list are implicitly ordered.

Why is this important?
Python dictionaries are a powerful and efficient datatype used to map one set of arbitrary objects to an associated but equally arbitrary set of objects.

Dictionary Operations

Python dictionaries contains several operations besides assigning and accessing elements. Some of the common operations are provided in the following table, but are is not an exhaustive list.

Accessing Dictionary Elements

To access all keys:
dictionary.keys()
To access all values:
dictionary.values()
To access a value:
dictionary['key']
To access all items (key-value pair) in dictionary:
dictionary.items()

Adding Key-Value Pairs

To add a key-value pair use:
dictionary["new_key"] = "new value"

Modifying Values

To modify a key-value pair use:
dictionary["key"] = "new value or modification"

Removing Key-Value Pairs

To remove a key-value pair use:
del dictionary["key"]

Looping Through a Dictionary

To loop through all key-value pairs:
for key, values in dictionary.items()
To loop through all keys:
for key in dictionary.keys()
To loop through all values:
for value in dictionary.values()
