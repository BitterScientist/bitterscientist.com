Split a PDF document into Individual Files

Background Info

Manual pdf manipulation is tedious and time intensive. Python has a PDF toolkit library PyPDF2 that can be used to:

  • extract document information
  • split documents by pages
  • merging documents page by page
  • merging multiple pages into a single page
  • cropping pages and much more.

Why is this important?
As data scientist and analysts, we are often called to extract data from various sources, included portable document format files. Knowing how to use the tools in this library will help automate the data extraction process, and file manipulation techniques that will save you time.

Goal
The goal of this post is to demonstrate how a pdf file can be split and saved into individual pages.

Case use example:
An example of its use is when you have received a file, where each page needs to be separated, and saved in a specific file directory. Instead of having to manually extract and save the file, this will carry out the entire process for you in seconds.
