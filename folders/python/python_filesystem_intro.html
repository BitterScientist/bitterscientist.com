Background Info

Filesystem: is the data structures (organization) that an operating system uses to keep track of files on a disk or partition.

  • A file system controls how data is stored and accessed.
  • Different operating systems have different filesystem conventions.
  • Windows uses backslashes to separate file or directory names, while forward slashes are used in UNIX/Linux/Mac.

Why is this important?
Working with files involves two basic activities:

  1. Reading and writing files
  2. Working with file systems: Creating, moving, deleting, or referencing to files

Python has several built in modules that enable users to carry out these tasks in a portable manner that is cross-platform compatible. Learning how to carry out these tasks are critical as a data scientist has to work with files.
