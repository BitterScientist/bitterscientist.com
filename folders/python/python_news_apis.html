Get News Data using Python

Background Info

News data can be a powerful source of information. Headlines in particular give you some insight into what society may view as important. There are several News APIs that allow you to get this information without having to manually extract it.

Why is this important?
Knowing how to extract news data using API calls is an important data science skills, as many projects may require you to conduct sentiment analysis on headlines, or look at trending topics in the news within a geographic range, or globally.

Goal
The goal of this post is to demonstrate how to use News APIs and transfer that data into a pandas dataframe for downstream analysis and applications.
