Introduction to Object Oriented Programming in Python

Background Info

Object oriented programming is one of the many ways to organize source code intended to perform the same set of functions, characteristics, and behaviors.

Why is this important?

  • Python is a common programming language used in data science.

  • Python libraries used in data science, such as pandas, numpy, matplotlib, scikit-learn all rely on the model and principles of object oriented programming.

  • Understanding the fundamentals of OOP is important to understand what is going on under the hood when libraries are used for an analysis.

Goal
The goal of this post is to provide an introduction to object oriented programming in python, and provide code examples of class construction, accessing instance and method, and using getters and setters.
