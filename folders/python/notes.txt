p class="general-p">
          Iterating through directories is a key part in file manipulation and automating certain actions, or finding a
          file type. Python provides several methods to traverse a directory, and includes:
          <ol>
            <li><code>os.listdir()</code></li>
            <li><code>os.scandir()</code></li>
            <li><code>os.walk()</code></li>
            <li><code>glob</code></li>
          </ol>
        </p>

        <p class="general-p"><strong>Why is this important?</strong><br>
