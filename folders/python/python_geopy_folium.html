Background Info

Geocoding: The process of transforming a physical address into latitudes and longitudes.

Why is this important?
Many mapping applications require latitudes and longitudes, and this geodata is often not provided in a dataset. Geopy and Geopandas libararies are tools Python uses to get this data.

Goals


The first goal of this project is to demonstrate how to obtain latitude and longitude from:

  • An address
  • Elements of an address (city, state, country)
  • Pandas Dataframe

The second goal is to demonstrate how these geocoordinates can be mapped using folium.

  • Using a for loop
  • Using dataframe.apply()
