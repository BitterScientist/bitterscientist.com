Use Python to Convert Word to PDF

Background Info

Converting word .docx files to pdf is tedious. Python has a library docx2pdf that can convert docx to pdf Windows or macOS directly using Microsoft Word (must be installed). DocumentationDocumentation describes installation steps. License: MIT License (MIT). Requires: Python >=3.5

Why is this important?
File format conversions are common place in various office settings. Manually converting these files is tedious, and learning how to automate this would save time, effort, and energy.

Goal
The goal of this post is to demonstrate how to convert word documents into pdf files.
