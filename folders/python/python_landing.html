Background Info

Guido van Rossum, 2006

Python: is a general purpose programming language that was created by Guido van Rossum, and initially released in 1991. Python was designed to be highly extensible, and built to allow developers to chose their coding methodologies. Python has become exceeding popular because its free, it's easy to learn, it's cross-platform compatible, it has a robust data science library, it's expressive, it's readable, and it has a large developer community. In fact, a 2017 stack overflow study showed that python has become the fastest-growing major programming language.

Why is this important?
The growth in its usage and popularity will make python an important skill for employers, especially enterprises interested in machine learning, and ai. Moreover, python is great for the development of small applications, and can scale well for large programs.
