Introduction to Python Census API: From Response to Pandas DataFrame

Background Info

The Census Bureau's mission is to serve as the nation's leading provider of quality data about its people and economy.
The data collects various types of data, and includes the following:

  • American community survey (ACS): Annual data that provides information on America's changing population, housing and workforce. Samples the country. website as of 12/6/20
  • Census of Governments: Identifies scope and nature of the nation's state and local government sector. website as of 12/6/20
  • Decennial Census of Population and Housing: The U.S. census counts each resident of the country, where they live on April 1, every ten years ending in zero. The Constitution mandates the enumeration to determine how to apportion the House of Representatives among the states. website as of 12/6/20
  • Economic Census: The U.S. Economic data. website as of 12/6/20

Why is this important?
The census is often used to understand trends in US population and economics. The census is a large publicsly available dataset that can be challenging to navigate. Knowing how to carry out an api call will make the data extraction and loading pipeline so much easier and an essential skill for an analyst and data scientist.

Goal

  • Demonstrate how to make an API call using python's response library.
  • Demonstrate and explain how to load the census API response payload into a pandas data frame that can be exported as a csv file.
