Background Info

Random numbers: is a number generated for, or part of, a set exhibiting statistical randomness. Python has a library that is dedicated to generating pseudo-random numbers.

Why is this important?
The python random library helps generate pseudo-random numbers that are used in various applications:

  • simulating new data when using bootstrapping methods
  • shuffling data before splitting into subsets
  • creating specific real-valued distributions, such as creating distributions of dataset that follows the gaussian distribution

Goal of this post
Is to use the random library to select an element from a list.
This will be accomplished by using a raffle as an example.
Where a function is built to select a name from a list of potential winners.
