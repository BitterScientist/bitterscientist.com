Background Info

Iterable Objects (Iterables): Are objects that can be used in a for loop, and includes:

  • list
  • tuple
  • set
  • dictionary
  • string
Iterable objects can implement iterator protocols, which are a class in Python that has the __next()__ method. This method provides the next value in an iterable object (list, tuple, set, dictionary, string), and it also keeps track of the state of the iterable.

Why is this important?
Iterable objects (and iterator protocols) are ubiquitous in programming, and they are help produce clean code, and under the hood, save a lot of computational resources.

Intro to iterators


An iterator is an object that implements a class in Python such as the __next()__ method. This method calls the next value in your iterable object when the next() function is called on it.
Python has many iterators, and all of the itertools functions return iterators.
