Background Info

Web Scraping: also known as web harvesting, and web data extraction is the process of gathering information embedded within the structure and style of websites.

Why is this important?
The world wide web is a rich data ecosystem, and it contains a seemingly unlimited supply of information. Web scraping enables analysts, data scientist and engineers the ability to gather information from the web. This is especially useful for websites that do not provide an API to gather access their data, or have restrictions. For detailed information on the legality of web scraping is beyond the scope of this post. Consult legal advice for more information.

What is covered in this post
Python has several libraries used for gathering information from the web. This post will demonstrate how to gather data in a table using the following libraries:

  • pandas: library used for performing data structure and data analysis tools. Scrapped data will be converted into a data frame, and then saved into a comma separated value (csv) file.
  • requests: library allows you to send an HTTP/1.1 requests
  • lxml.html: library is an HTML parser
