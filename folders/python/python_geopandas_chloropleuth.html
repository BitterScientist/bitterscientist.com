Visualize Geographic Information Systems (GIS) Data using Python's Cloropleuths and Folium.



Background Info

Geographic Information Systems (GIS) Data: is a system for capturing, storing, manipulating, and displaying data related to positions on Earth's surface. Python has several libraries dedicated to working with GIS data.

  • Geopandas: Pandas with GIS data processing capabilities.
  • GeoPy: locate the coordinates of addresses, cities, countries, and landmarks across the globe using third-party geocoders and other data sources.
  • Arcpy: geoprocessing operations, spatial analysis, data conversion, and management and map production with Esri ArcGIS.
  • GeonamesCache: retrieve names, ISO and FIPS codes of continents, countries as well as US states and counties as Python dictionaries. The country and city datasets also include population and geographic data.
  • GDAL/OGR: translating between GIS formats and extensions
  • PyProj: project and transform coordinates with a range of geographic reference systems.

Why is this important?
Geographic Information Systems data is an important part of the data ecosystem, and is important at understanding spatial patterns and relationships. For example, during the 2020 covid pandemic, contact tracing relies on GIS data.

The Objective
The objective of this post is to demonstrate how Python can be used to:

  • Load and manipulate GIS data
  • Create basic visualization of GIS data using Matplotlib
  • Create Cloropleuths (maps used to represent statistical data by using shading patterns on geographic areas)
  • Combine a Cloropleuth with Folium, an interactive map.

Data source
The data for this project was obtained from the LA County GIS Data Portal, and consists of the 2010 Census Information for LA County.

Methods overview

  1. Import data using geopandas.read_file()
  2. Explore data datatypes, coordinate reference system, shape
  3. Feature Engineering Add feature (area)
  4. Create simple visualization of census tracts Matplotlib.pyplot
  5. Merge census tract data with median household income using DataFrame.merge()
  6. Create a Cloropleuth using the median income feature as the shading indicator for the map
  7. Create an interactive map that would provide greater detail with the Cloropleuth using the folium library.
