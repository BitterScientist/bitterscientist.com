Use Python to Create or Delete Directories.

Background Info

File management can be tedious, especially if it involves creating or removing directories. All this can be done directly in a jupyter notebook.

Why is this important?
Knowing how to use the os module is an essential skill that can help you as you manipulate files directly through jupyter notebook.

Goal
The goal of this post is to demonstrate how to create and delete directories directly from a jupyter notebook.
