Objective¶

Demonstrate how to create figures with multiple subplots in python using plt.subplots()¶

  • Create various subplot grids
  • Add spacing around subplots
  • Add data to subplots
  • Add labels
  • Customize plots
  • Use for loops to create subplots

Data¶

  • UNAIDS is leading the global effort to end AIDS as a public health threat by 2030 as part of the Sustainable Development Goals.
  • The data represents the prevalence of HIV, total (% of population ages 15-49)
  • The data in this example was obtained from https://data.worldbank.org/indicator/SH.DYN.AIDS.ZS?view=chart
  • Data retrieved 1-18-20
  • Some non-country rows were removed from the csv file prior to import

Why use plt.subplots()¶


A convenient matplotlib method to render a grid of subplots. This approach creates a new figure and returns a numpy array with the subplot objects.
1 Gives you handle of the axes. This will allow you to index array of axes like a 2d array.

2 Shared axis options. You can define if the subplots will have the same x- or y-axis using the sharex and sharey arguments.
  This will allow you to compare data on the same scale.

Key plt.subplots() Options:¶

Argument Detail
nrows int - Defines the number of rows of subplots
ncols int - Defines the number of columns of subplots
sharex Boolean - Defines if all subplots will have the same x-axis ticks and range
sharey Boolean - Defines if all subplots will have the same y-axis ticks and range
subplot_kw dictionary - keywords used to create each subplot

Add spacing around subplots using subplots_adjust()¶

  • matplotlib.pyplot.subplots_adjust(left=None, bottom=None, right=None, top=None, wspace=None, hspace=None)
  • wspace and hspace control the space, in percent, of the figure width and height.
  • use floats to adjust spacing
  • documentation: https://matplotlib.org/api/_as_gen/matplotlib.pyplot.subplots_adjust.html

    axes.plot(years, four_countries[country])

    axes.set_title(country)

How does this work?!¶

fig, axes = plt.subplots(2, 2, figsize=(15,10))

  • This code will create a figure object and set the figure size at width 15 and height 10
    for axes, country in zip(axes.ravel(), country_names):
  • the country_names is a list of the countries we will plot
  • since we are using a for loop, country is an index of the country_names array that as we iterate through the for loop, will change country
  • the .ravel() in axes.ravel() is a numpy method that returns a contiguous flattened array. axes is 2d [row index, col, index]
  • the zip() method helps us interate through the axes and country_names list as each subplot is created during the for loop.
    ax.plot(years, four_countries[country])
  • This code will add the specific country data to the plot in the current loop
    ax.set_ylabel('Prevalence of HIV, \n total (% of population ages 15-49)')
    ax.set_xlabel('Years')
  • This code adds the labels to the subplot in the current loop
    ax.set_xticklabels(years, rotation=45)
  • This code will rotate the x-tick labels so they are visible!
    ax.set_title(country)
  • This code will add the title of the current subplot in the current loop
