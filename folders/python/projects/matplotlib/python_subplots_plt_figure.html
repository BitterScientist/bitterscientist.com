  • Data retrieved 1-18-20

Creating subplots using plt.figure()¶

  • matplotlib.pyplot.figure(num=None, figsize=None, dpi=None, facecolor=None, edgecolor=None, frameon=True, FigureClass, clear=False, **kwargs)
  • Documentation: https://matplotlib.org/api/_as_gen/matplotlib.pyplot.figure.html

Why use plt.figure()¶


1 Gives you handle to a figure. This will allow you to store this instance in a variable and use it later on.
  Example:
  fig = plt.figure()
  #... other code
  fig.autofmt_xdate()

2 Set figure parameters. You can set some parameters for the figure by supplying them as arguments to plt.figure.
  Example:
  plt.figure(figsize=(10,10), dpi=200):


3 Create multiple figures. You can create several figures in the same script.
  Example:   plt.figure() # create a figure
  plt.plot([2,2,1])
  plt.figure() # create another figure
  plt.plot([2,2,2]) # successive commands are plotted to the new figure

Key plt.figure Options¶

  • num : integer or string, optional, default: None
  • figsize : (float, float), optional, default: None. Gurantees the figure will be a certain size when saved to disk.
  • dpi : integer, optional, default: None. Resolution of the figure.

Add subplot(s) using add_subplots(nrows, ncols, index)¶

  • nrows: Number of rows that will contain subplots in the figure
  • ncols: Number of columns that will contain subplots in the figure
  • index: Label of the returned index
    The number of subplots is nrows * ncols

Example: Create a 2x2 figure with 3 plots¶

In [10]:

fig = plt.figure()
# To create a subplot using this method, you need to create a unique axes instance variable and assign the 
# subplot to that axes
ax1 = fig.add_subplot(2, 2, 1) #This will create a plot on the top left grid
ax2 = fig.add_subplot(2, 2, 2) #This will create a plot on the top right grid
ax3 = fig.add_subplot(2, 2, 3) #This will create a plot on the bottom left grid
plt.show()

How does this work?!¶

fig = plt.figure(figsize=(15,10))

  • This code will create a figure object and set the size 15 width by 10 height
    for country, num in zip(country_names, range(1,5)):
  • the country_names is a list of the countries we will plot
  • since we are using a for loop, country is an index of the country_names array that as we iterate through the for loop, will change country
  • num is also a counter, although it is counting the index of the subplot
  • range(1,5) indicates how many subplots, range has to be >0, and the last number is not inclusive
  • zip(country_names, range(1,5) is getting the country index and range index for us that will be used inside the for loop
    ax = fig.add_subplot(2,2, num)
  • notice that here we are creating two rows and 2 columns, and the last position is 'num', as we move through the for loop, we will start at num = 1, so the first subplot will be first row, first column
    ax.plot(years, four_countries[country])
  • This code will add the specific country data to the plot in the current loop
    ax.set_ylabel('Prevalence of HIV, \n total (% of population ages 15-49)')
    ax.set_xlabel('Years')
