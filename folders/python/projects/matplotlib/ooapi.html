Objective:¶

Provide an introduction to data visualization using matplotlib by:

  • Introducing the object oriented api interface
  • Describe the anatomy of a figure object
  • Demonstrate how to create a basic line plot
  • Review how to customize plots
  • Show how to save images Matplotlib documentation can be obtained here:
    https://matplotlib.org/tutorials/introductory/lifecycle.html

Data:¶

  • The data used for this post were taken from the South Coast AQMD website.
  • The South Coast Air Quality Management District (AQMD) develops plans and regulations designed to achieve public health standards by reducing air pollution from business and industry. In these datasets, the pollutant is Ozone (O3)
  • SCAQMD provides 6 months of historic data for various regions it oversees
  • https://xappprod.aqmd.gov/aqdetail/AirQuality/HistoricalData
  • The data from two areas were selected for this study include Central LA, and Northwest Coastal LA County.
  • Reporting Period July 31, 2019 -Jan 1, 2020
  • Data retrieved 1-15-20

The object-oriented (OO) interface¶

The OO interface allows users to prepare figures with many subplots.

Two instances are used in this method:

  • 1) axes.Axes - represents an individual plot do not confuse with "axis", which refers to the x/y axis of a plot.
  • 2) figure.Figure: is the final image that may contain 1 or more Axes.
  • the figure is the canvas, and the axes instances are the images on the canvas.

  • matplotlib.pyplot.plot(*args, scalex=True, scaley=True, data=None, **kwargs)
  • Documentation https://matplotlib.org/3.1.1/api/_as_gen/matplotlib.pyplot.plot.html
  • This Plot y versus x as lines and/or markers.
  • plot(x, y) # plot x and y using default line style

color selection¶

Colors https://matplotlib.org/api/colors_api.html

  • Some formats to specify a color:
  • GB or RGBA tuple of float values in [0, 1] (e.g., (0.1, 0.2, 0.5) or (0.1, 0.2, 0.5, 0.3))
  • Hex RGB or RGBA string (e.g., '#0f0f0f' or '#0f0f0f80'; case-insensitive)

Adding a title¶

Set plot title using .set_title() https://matplotlib.org/3.1.1/api/_as_gen/matplotlib.axes.Axes.set_title.html
Important Arguments:

  • label - Text to use for the title
  • fontdict - A dictionary controlling the appearance of the title text, the default fontdict is:
  • loc : {'center', 'left', 'right'}, str, optional the "\n" will give you a line break :)
In [17]:
# multiple lines with multiple colors
fig, ax = plt.subplots()
ax.plot(month_nwcoast_df.index.values, month_nwcoast_df['Value'])
ax.set_xlabel("Time (months)")
ax.set_ylabel("Ozone (pphm)")
ax.set_title(label="Monthly Ozone levels \n in the Northwest Coast of Los Angeles", loc='center')
plt.show()
