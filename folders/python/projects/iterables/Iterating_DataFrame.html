Transversing a DataFrame¶

There are several methods to transverse a dataframe, several of which use a generator object:

  • for loop (column names)
  • iteritems(): generator object transverses through the columns
  • iterrows(): generator object transverses through the rows

import pandas as pd

Iterate over each element of the dataframe, row-wise using iterrows()¶

Create a generator using .iterrows()¶

In [5]:
# Apply iterrows() methods on only a subset of the data
data = fighters[0:2].iterrows()
In [6]:
# Confirm generator type
type(data)
Out[6]:
generator
In [7]:
# Confirm generator object
print(data)

<generator object DataFrame.iterrows at 0x1116ea8d0>

Access each instance as a tuple in the dataframe using .itterrows() and for loop¶

  • Each item that is returned/transversed will be a tuple (index name, Series)

    print(item)
