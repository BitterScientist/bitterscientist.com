Goals:¶

Introduce methods used to get information on files and directories using path-information functions:

  • The os.path.exists() method
  • The os.path.isdir() method
  • The os.path.isfile() method

Import dependencies¶

In [1]:
import os

Test if path exists using the os.path.exists() method¶

  • Returns True if path exists, otherwise False
In [2]:
# Path exists
os.path.exists('/Users/trinitycisneros/Documents')
Out[2]:
True
In [3]:
# Path does not exist
os.path.exists('/Users/trinitycisneros/Documents/Imaginary')
Out[3]:
False

Test if directory exists using the os.path.isdir() method¶

  • Returns True if directory exists, otherwise False
In [4]:
# Directory exists
os.path.exists('/Users/trinitycisneros')
Out[4]:
True
In [5]:
# Directory does not exist
os.path.exists('/Users/trinitycisneros/imaginary')
Out[5]:
False

Test if file exists using the os.path.isfile() method¶

  • Returns True if file exists, otherwise False
In [6]:
# File exists
os.path.isfile('/Users/trinitycisneros/Downloads/test.pdf')
Out[6]:
True
In [7]:
# File does not exist
os.path.exists('/Users/trinitycisneros/Downloads/test.png')
Out[7]:
False
