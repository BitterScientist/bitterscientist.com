Goals:¶

Introduce Python pathname manipulation methods using:

  • The os method os.path.join()
  • The pathlib method path()
  • Additional os methods used to split, or extract pathname components

Manipulate pathnames using os.path.join()¶

  • Action: Join one or more path components intelligently.
  • Why this is cool::
  • Method lets you form file paths from a sequence of directory or file names independent of the conventions underlying the operating system.
    Documentation: https://docs.python.org/3/library/os.path.html
  • Example of method when used in Windows
  • Example of method when used in Linux/UNIX
  • Example of method when used in Mac

Import Dependencies¶

In [1]:
import os

Construct pathname using os.path.join()¶

  • Arguments can be single directory, filenames, and subpaths

Windows Example 1¶

Code:
print(os.path.join('bin', 'utils', 'disktools'))
Output:
bin\utils\disktools

Windows Example 2 with subpaths¶

Code:
print(os.path.join('mydir\\bin', 'utils\\disktools\\chkdisk'))
Output:
mydir\bin\utils\disktools\chkdisk

Linux/UNIX/Mac Example 1¶

In [2]:
print(os.path.join('bin', 'utils', 'disktools'))

bin/utils/disktools

Linux/UNIX/Mac Example 2 using variables¶

In [3]:
path1 = os.path.join('mydir', 'bin')
path2 = os.path.join('utils', 'disktools', 'chkdisk')
print(os.path.join(path1, path2))

mydir/bin/utils/disktools/chkdisk
Comment:
 Windows system uses backslahses to join path components
Linux/UNIX system uses forward slashe separators to join path components

Manipulate pathnames using the pathlib method path()¶

  • Action: Join one or more path components intelligently.

Why this is cool::

  • Method lets you form file paths from a sequence of directory or file names independent of the conventions underlying the operating system.
  • Pathlib is object oriented method for pathname manipulation
    Documentation: https://docs.python.org/3/library/pathlib.html

Import dependencies¶

In [4]:
from pathlib import Path

Windows Example¶

Code:
current_path = Path() print(current_path.joinpath('bin', 'utils', 'disktools'))
Output:
bin\utils\disktools

Linux/UNIX/Mac Example¶

In [5]:
# Instantiate Path() method
current_path = Path()
In [6]:
# Evoke joinpath() method
print(current_path.joinpath('bin', 'utils', 'disktools'))

bin/utils/disktools

Use os.path.split() to split path¶

What it does:

  • Split the pathname path into a tuple, (head, tail) where tail is the last pathname component and head is everything leading up to that.
In [10]:
# path 
path = '/home/User/Desktop/tail.txt'
  
# Split the path in  
# head and tail pair 
head_tail = os.path.split(path) 
  
# print head and tail 
# of the specified path 
print("Head of '% s:'" % path, head_tail[0]) 
print("Tail of '% s:'" % path, head_tail[1], "\n") 

Head of '/home/User/Desktop/tail.txt:' /home/User/Desktop
Tail of '/home/User/Desktop/tail.txt:' tail.txt

Use os.path.basename() to return basename of path¶

What it does:

  • Method returns only the basename of the path.
In [11]:
# path 
path = '/home/User/Desktop/tail.txt'
  
# Split the path in  
# head and tail pair 
basename = os.path.basename(path) 
  
# print output
print("The basename of the path is '% s:'" % basename) 

The basename of the path is 'tail.txt:'

Use os.path.dirname() to return the path up to, and not including, the basename¶

What it does:

  • Method returns the path before the basename.
In [12]:
# path 
path = '/home/User/Desktop/tail.txt'
  
# Split the path in  
# head and tail pair 
dirname = os.path.dirname(path) 
  
# print output
print("The dirname of the path is '% s:'" % dirname) 

The dirname of the path is '/home/User/Desktop:'
