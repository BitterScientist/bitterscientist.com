Create Directory using os.mkdir()¶

In [2]:
path = 'new_directory' #The name of the directory that will be created
try:
    os.mkdir(os.path.join(os.getcwd(), 'new_directory'))
except OSError:
    print ("Creation of the directory %s failed" % path)
else:
    print ("Successfully created the directory %s " % path)

Successfully created the directory new_directory

Code Explained¶

  • Line 3: Create the file path for where the new directory will be stored, and then creates it with os.mkdir().
    • os.mkdir(): Create a directory named path with numeric mode mode.
    • Documentation: https://docs.python.org/3/library/os.html
    • os.getcwd(): Gets the current directory path.
    • os.path.join(): Joins together the arguments into a single path.
  • Line 2, 4, 6 try except else: Handles exceptions and prints out messages based on which conditions were met.
  • Documentation: https://docs.python.org/3/tutorial/errors.html
