import os¶

  • Documentation: https://docs.python.org/3/library/os.html
  • What it does: provides a portable way of using operating system dependent functionality.
  • What we are using it in this post::
    • os.path.join(): create filepath
    • os.getcwd(): get current working directory file path
    • os.listdir(): get a list of files in a specific directory
    • os.path.basename(): returns a string that contains the base name the specified path.

import glob¶

  • Documentation: https://docs.python.org/3/library/glob.html
  • What it does: finds all the pathnames matching a specified pattern according to the rules used by the Unix shell.
  • What we are using it in this post::
    • glob.glob(): get all the files in a specific directory that have a specific .docx extension.

import shutil¶

  • Documentation: https://docs.python.org/3/library/shutil.html
  • What it does: offers a number of high-level operations on files and collections of files. In particular, functions are provided which support file copying and removal.
  • What we are using it in this post::
    • shutil.move(): Move files.

import re¶

  • Documentation: https://docs.python.org/3/library/re.html
  • What it does: provides regular expression matching operations similar to those found in Perl.
  • What we are using it in this post::
    • re.match(): match specific portion of filename.
