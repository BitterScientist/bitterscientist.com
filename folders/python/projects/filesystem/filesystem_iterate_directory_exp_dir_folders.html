List Comprehensions:

  • generates a list containing the folders that will be traveresed as the pdf merging function is executed.

  • os.listdir(directory): returns a list of items in the directory filepath

  • f for f in os.listdir(directory): iterates through each item and will save into new list
  • if not f.startswith('.') conditional statement, will exclude any item in directory that starts with '.' These files are hidden.
