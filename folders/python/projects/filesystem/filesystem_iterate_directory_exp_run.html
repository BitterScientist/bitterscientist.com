  • Evokes the function and passes in the required parameters
  • directory the directory that contains the folders that will be traversed
  • folders list of folders that contain files that will be merged
  • '*suffix_*.pdf' the pattern that is to be matched in the file selection to be merged.
  • 'pdf_merged.pdf' the name of the file that will be saved when pdfs are merged
