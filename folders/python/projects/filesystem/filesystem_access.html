Get list of files in working directory with os.listdir(os.curdir)¶

  • os.curdir(): Refers to the current directory.
  • os.listdir(): Method used to get the list of all files and directories in the specified directory.
In [3]:
os.listdir(os.curdir)
Out[3]:
['test_folder', '.DS_Store', 'intro_filesystem.ipynb', '.ipynb_checkpoints']

Change Working Directory with os.chdir()¶

  • os.chdir('folder name'): Change the current working directory to specified path
In [4]:
os.chdir('test_folder')
os.getcwd()
Out[4]:
'/Users/trinitycisneros/Documents/Coding/bitterscientist.com/bitterscientist.com/folders/python/projects/filesystem/test_folder'
