Selection Function¶

  • Takes in a list of elements, the number of selections, and an empty list that will be used to store selection
In [4]:
def select_winner(names, num_winners, winners_list):
    for i in range(0, num_winners):
        winner = random.randint(0, len(names))
        if names[winner] not in winners_list:
            winners_list.append(names[winner])
        else:
            next
    return winners_list

Explanation¶

def select_winner(names, num_winners, winners_list):

  • function that takes in three arguments
  • names list of elemets
  • num_winners number of elements to select as an integer
  • winners_list an empty list that will be used to store the selected element(s)

    for i in range(0, num_winners):
  • for loop that will iterate through the list of names by the number of selections, or winners in this example, made.

    winner = random.randint(0, len(names))
  • this code will then use use the random.randint() method to select a random number between 0 - the total number of element sint he list names.
  • this number will be saved as winner and will be used as an index (position) for the name selection

    if names[winner] not in winners_list: winners_list.append(names[winner])
  • this block of code has a conditional statement that is important in cases where more than 1 element is selected (more than one winner).
  • the random number selected and assigned to winner is used with the names list as the index, and the name of that winner/element is then returned
  • the if statement then checks if that name is in the winner_list.
  • if it is not already selected and stored in the winner_list, then it will be execute the next line of code, and it will add the name/element to the winner_list using the the .append() method.
    else: next
  • if the if names[winner] not in winners_list: is false, then the winner is already in the list, then this code tells is to skip that number and return to the for loop.

    return winners_list
  • when the for loop terminates, then this return statement will return the list of selected elements
