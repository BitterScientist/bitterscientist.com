Data¶

Objective¶

The goal is to clean the dataset for analysis.

  • Clean up table header by removing redundant substrings

    • Method 2: string.replace(old, new)

Method 1: Modify Strings using Python string.lstrip()¶

string.lstrip(s[, chars])

  • Return a copy of the string with leading characters removed. If chars is omitted or None, whitespace characters are removed. If given and not None, chars must be a string; the characters in the string will be stripped from the beginning of the string this method is called on.
  • The string does not have to be identical.
  • You can chain string.lstrip() methods string.lstrip('string1').lstrip('string2')
  • Documentation: https://docs.python.org/2/library/string.html

Method 2: Modify Strings using Python string.replace()¶

string.replace(s, old, new[, maxreplace])

  • Return a copy of string s with all occurrences of substring old replaced by new. If the optional argument maxreplace is given, the first maxreplace occurrences are replaced.
  • You can chain commands string.replace(s, old, new[, maxreplace]).replace(s, old, new[, maxreplace])
  • Documentation: https://docs.python.org/2/library/string.html
