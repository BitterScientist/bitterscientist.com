Function 3: Calculate the Tangent Line¶

def tangent_line(f,x_0, a, b): x = np.linspace(a,b,n) y = f(x) y_0 = f(x_0) y_tan = deriv(f,x_0) * (x - x_0) + y_0 return x, y_tan

  • Paramters:
    • f1 is a function
    • x_0 is some number along the x axis.
    • a, b the start value that will be input into numpy.linespace(), which returns an evenly spaced numbers between an interval. a is the start and b is the end of that interval
    • Documentation for linespace: https://numpy.org/doc/stable/reference/generated/numpy.linspace.html
  • What it does:
    • x = np.linspace(a,b,n): creates an array with n number of evently spaced values between a and b.
    • y = f(x): creates an array with the derivatives of each value in x
    • y_0 = f(x_0): returns the derivative of the new x value that will form the tangent line
    • y_tan = deriv(f,x_0) * (x - x_0) + y_0: returns an array of y tangent values
    • return x, y_tan: returns the x and y_tan value at end of function.
