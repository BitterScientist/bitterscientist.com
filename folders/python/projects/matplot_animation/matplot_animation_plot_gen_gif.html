Set-up Data to Plot¶

f = list(range(-4, 4))

  • What it does:
    • Creates a list of numbers between -4 and 4
    • This will be used for the x_0 in the tangent line function
    • The lenght of values is also the number of tangent lines that will be rendered.

ax.plot(np.linspace(-4, 4, 100), np.linspace(-4, 4, 100)**2, lw=4)

  • What it does: Create the parabola line that is blotted in blue. The tangent lines will be drawn against this curve.
  • np.linspace(-4, 4, 100): data for x values: this will create an array of values between -4 and 4 with 100 evently spaced numbers in between.
  • np.linspace(-4, 4, 100)**2: data for y values: this will create an array of values between -4 and 4 with 100 evently spaced numbers in between, and each value will be the square.
  • lw=4: How thick the line will be for the parabola.

xdata, ydata = [], [] ln, = plt.plot([], [], 'r', animated=True) annotation = ax.text(-0.5, 4, '')

  • xdata, ydata = [], []
    • What it does: Create an numpy arrays which will contain the frames at which we will draw our animation.

  • ln, = plt.plot([], [], 'r', animated=True)
    • What it does: Create a line that is red that will be animated

  • annotation = ax.text(-0.5, 4, '')
    • What it does: Creates the text box that will be positioned at -0.5, 4, and is empty

Function to Initialize the plots¶

def init(): ax.set_xlim(-4, 4) ax.set_ylim(-5, 20) ln.set_data(xdata,ydata) annotation.set_text('') return ln, annotation

  • ax.set_xlim(-4, 4) ax.set_ylim(-5, 20)
    • What it does: it sets the plot dimensions according to the x and y limits
  • ln.set_data(xdata,ydata)
    • What it does: creates an empty line using xdata and ydata parameters that were instantiated during the set-up stage
  • annotation.set_text('')
    • What it does: sets up the empty string of text that will be updated with information as the plot is animated.
  • return ln, annotation
    • What it does: Returns the initialized line and annotation objects

Function to animate (update) the plots¶

def animate(i): x, y_tan = tangent_line(f1,i,-4,4) ln.set_data(x, y_tan) annotation.set_text('y = %.1f' % (-1*y_tan[i])) return ln, annotation

  • x, y_tan = tangent_line(f1,i,-4,4)
    • What it does: it calls on the tangent_line() function and returns the x and y_tan value.
  • ln.set_data(x, y_tan)
    • What it does: sets the x and y_tan value as the parameters for the line that will be plotted.
  • annotation.set_text('y = %.1f' % (-1*y_tan[i]))
    • What it does: it gets the information from the tangent y value and then adds that as a string to the text that will be plotted and updated with each new y_tan value..
  • return ln, annotation
    • What it does: Returns the updated line and annotation objects

Call FuncAnimation and show¶

anim = animation.FuncAnimation(fig, animate, init_func=init, frames=f, interval=0, blit=True)

  • What it does: Makes an animation by repeatedly calling a function func.
  • Documentation: https://matplotlib.org/3.3.0/api/_as_gen/matplotlib.animation.FuncAnimation.html
  • Parameters:
    • fig: The figure object used to get needed events, such as draw or resize.
    • animate: The function to call at each frame (the update or animate function we created).
    • init_func: A function used to draw a clear frame. We created this, but its optional.
    • frames: Source of data to pass func and each frame of the animation. In this function, only 9 frames from f = list(range(-4, 4)).
    • interval: Delay between frames in milliseconds.
    • blit: Whether blitting is used to optimize drawing.

Save as Gif¶

anim.save('parabola_der_animation.gif', writer='imagemagick', fps = 10)

  • What it does: method used to save animation.
  • Documentation: https://matplotlib.org/3.3.0/api/_as_gen/matplotlib.animation.Animation.html?highlight=animation%20save#matplotlib.animation.Animation.save
  • Parameters:
    • 'parabola_der_animation.gif' file name with extenstion
    • writer='imagemagick specifies how it will be saved. For GIFs, will need to install imagemagic
    • Documentation: https://docs.wand-py.org/en/0.3.5/guide/install.html
    • fps = 0 Movie frame rate (per second). If not set, the frame rate from the animation's frame interval. Optional Field. Here 0 because not a movie, but a gif that will be saved.
