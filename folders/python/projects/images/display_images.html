Three methods to display grids of images inside jupyter notebook using Python

    Post Goals, Demonstrate how to display a grid of images in jupyter notebook using:
  1. IPython.display HTML and Image.

  2. The ipyplot library

  3. Matplotlib

    Motivation:
  • Jupyter notebook is a popular web-based interactive environment to display, share documents that contain code, equations, and visualizations.

  • Displaying data visualizations using python libraries such as python, matplotlib, and seaborn is well documented, but there are many cases where displaying images can be helpful.

  • Images can be added using from local files and a url, although most methods only allow you to display one image, in some cases you may want to display a grid of images. Grids allow you to compress your images in a well contained cell without having to provide multiple cells that take up space.

  • The aim of this post is to demonstrate how to display a grid of images using a few common and well documented methods.

Display a grid of images using iPyPlot


    ipyplot
  • Simple package that leverages IPython and HTML for more efficient, reach and interactive plotting of images in Jupyter Notebooks.

  • Additional information can be found here: Documentation as of date of this post.

  • You will need to install this package.

  • iPyPlot is a quick and efficient method for plotting images inside a Python Notebook cell and combines HTML and iPython for you. It contains a few image display objects, but in this case, we will only demonstrate how to display a collection of images.

  • Why is this an improvement? It doesn't require you to hard code the html table code or use a for loop, subplots, or any other code to render your grid. The only downfall is that it display the local or remote image file url, but this isn't a big deal.
    Process
  • Import ipyplot.

  • Prepare a list of labels, and they should be a sequence of strings or int.

  • Prepare a list of images. The list should be a sequence of either strings (local or remote image file URLS), PIL.Image objects or numpy.ndarray objects representing images.

  • Use this code: ipyplot.plot_images(images, labels, img_width=width).

  • the width is an integer.

Display a grid of images using matplotlib


    matplotlib
  • Matplotlib is a comprehensive library for creating static, animated, and interactive visualizations using Python.

  • Additional information can be found here: Documentation as of date of this post.

  • You will need to install this package.

  • This method provides greater control over image display, labeling, and formatting.

  • Greater control means more coding, and the following steps will show you how to displays a 3x2 grid of images horizontally or vertically
    Process
  • Import modules.

  • urllib is a pacakge that collects several modules for working with urls.

  • matplotlib imshow displays data as an image on a 2d regular raster.

  • matplotlib imread reads an image from a file into an array.

  • _, axarr = plt.subplots(3, 2, figsize=(10,10)): sets up your subplot and figure size

  • curr_col = 0: sets up a column counter

  • for index in range(len(images)): for loop that will plot the number of images that will be plotted.

  • image_path = plt.imread(images[index]) Will fetch the image url at the index and use matplotlib read() to save the image data as an array.

  • row = index % 3: This code uses a modulus to track the position of the row. When the row is 3, it will reset to 0.

  • axarr[row,curr_col].imshow(image_path): code uses matplotlib imshow() to display the image at the subplot position [row, curr_col].

  • if row == 2:: when row is 2, the curr_col will increase by one. Row starts at 0, and increments at each for loop. It is reset when the for loop hits 3. We are using this method because we will plot 3 rows and 2 columns. This is a vertical long layout. Images are plotted in column 1 and go down to row 3 (or 2 since we are counting from 0, 1, 2). Once we add the image in row 3 (identified as 2), then the current column will increment by 1, and we will then add the next image in column 2, row 1.

  • curr_col += 1: this is the code that will increment the curr column.

  • The last two lines of code are used to place the images closer and plt.show() displays all open figures.

# Vertical 3 x 2
_, axarr = plt.subplots(3, 2, figsize=(10,10)) #two rows and #3 columns
curr_col = 0
for index in range(len(images)):
     # fetch the url as a file type object, then read the image
        image_path = plt.imread(images[index])
        
        # find the row by taking the current index modulo 2
        row = index % 3
        
        # plot on relevant subplot
        axarr[row,curr_col].imshow(image_path)
        if row == 2:
            # increment row
            curr_col += 1

plt.tight_layout()          
plt.show()
