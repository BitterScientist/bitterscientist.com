Geocoding addresses from Pandas dataframe¶

Import Data¶

  • Consists of a single column with city state and country as a string
In [7]:
df = pd.read_csv("city_state_country.csv")
In [8]:
# Create a geolocator that holds the Geocoding service
geolocator = Nominatim(user_agent="myGeocoder")
  • Delay Geocoding in second between each address using Ratelimiter()
  • Important when Geocoding a large number of physical addresses as the Geocoding service provider can deny access to the service.
In [9]:
# RateLimiter allows to perform bulk operations while gracefully 
# handling error responses and adding delays when needed.
geocode = RateLimiter(geolocator.geocode, min_delay_seconds=5)
In [10]:
# Create a df['location'] column by applying geocode we created
df['location'] = df['city_state_country'].apply(geocode, timeout=10)
In [11]:
# Third, we can create latitude, longitude, and altitude as a single tuple column
df['point'] = df['location'].apply(lambda loc: tuple(loc.point) if loc else None)
In [12]:
# split point column into latitude, longitude and altitude columns
df['lat'], df['lon'], df['altitude'] = df.point.str
In [13]:
# Remove unnecessary columns
df.drop(['location', 'point'], axis=1, inplace=True)
