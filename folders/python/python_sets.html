Working with Python Sets

First Posted: 2-7-2020
Last Updated: 2-9-2020

Background Info

Python Sets: are an unordered collections of unique objects that are immutable and hashable. Sets can contain ints, floats, strings, and tuples, but sets cannot contain dictionaries, lists, and other sets.

Why is this important?
Sets are an important data type that are often used when membership and unique values are important. This means sets are powerful tools that help remove a) duplicate values, and b) perform mathematical operations such as set theory (intersections, unions, difference, and symmetrical difference.

Creating Sets

To create a set:
set_variable = set([list])

Adding and Removing Elements in a Set


Add an element to a set:
set.add(element)
Remove an element from a set:
set.remove(element)

Set Collection Methods

Check if element in set:
element in set
Return number of elements in a set:
len(set)

Looping Through a Set

You can loop through the elements in a set using the same method for lists.
for element in set:

dosomething with element

Set Operation: Union


Combines two sets:
Option 1: set_a | set_b Option 2: set_a.union(set_b)

Set Operation: Intersection

Returns elements that two sets have in common.
Option 1: set_1 & set_2 Option 2: set_1.intersection(set_2)

Set Operation: Difference

Returns the elements that are in one set, and not the other
This method will return items in set_1 that are not in set_2
To get the elements in set_2 not in set_1, reverse order.
Option 1: set_1 - set_2 Option 2: set_1.difference(set_2)

Set Operation: Symmetric Difference

Returns all values that are not in common between two sets
Option 1: set_1 ^ set_2 Option 2: set_1.symmetric_difference(set_2)
