How to connect to Socrata and Import Data to Pandas using Jupyter Notebook.

Background Info

Access to publicly available datasets is essential to any data sleuth. Major sites have established APIs to help with the data download. The Socrata Open Data API allows you to programmatically access open data from governments, non-profits, and NGOs around the world.

Why is this important?
Knowing how to access data from the Socrata Open Data API will open up an entire universe of free data for your projects.

Goal
The goal of this post is to demonstrate how to
1) get data from the Socrata API using unauthentical and authenticated methods.
2) read the data from JSON to Pandas.
Note: these methods may have chanced since this posts publication on 7/19/20, please consultation official documentation.
