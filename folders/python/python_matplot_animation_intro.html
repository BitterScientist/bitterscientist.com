Create Animated Plots Using Matplotlib and Python

Background Info

Matplotlib provides easy methods to generate animated plots.

Why is this important?
Data visualization is an important step in any analysis. Learning how to render animated plots can provide additional insights that might be lost in a static image.

Goal
The goal of this post is to demonstrate how to create an animated plot in a Graphics Interchange Format (GIF). In this example, a parabola will be displayed and several tangent lines will be animated.
