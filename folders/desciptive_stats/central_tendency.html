Posted: 12-7-2019

Last Updated: 12-7-2019

Background Info

Carl Friedrich. One of the first scientist to suggest that the most probable value in a dataset can be represented by the arithmetic mean.

Central tendency: describes where most of the data is located in its range of values. Also known as: Average, center, location. The center tendency of a one dimensional dataset can be examined using various measures (formulas), and which one is used depends on the data being analyzed.

Why is this important?
Understanding the central tendency of a dataset is one of the first steps in exploratory data analysis, and is often one of the first reported features of a data set. Most people are familiar with the average, but there are many other measures of central tendency, and this post will explore them in detail.

Arithmetic Mean (Mean), "Average"

Definition: The sum of all values divided by the number of values.
When to use:
R script Python Excel

Weighted Mean, "Weighted Average"

The sum of All values that were multiplied by some "weight". The total sum is then divided by the sum of the weights. When to use:
R script Python Excel

Median

Definition: The position in the dataset where half of the values are below, and above. When to use:
R script Python Excel

Weighted Median

Definition: The position in the dataset where half of the weighted values are below, and above. When to use:
R script Python Excel

Trimmed Mean, "Truncated Mean"

Definition: A mean that is obtained after dropping a fixed number of extreme values. When to use:
R script Python Excel

Mode

Definition: The most commonly occurring value, or category in a dataset. When to use:
R script Python Excel
