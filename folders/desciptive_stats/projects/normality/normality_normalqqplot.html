Introduction to Testing for Normality.

    Intro to Testing for Normaility

  • In statistics, testing for normality is used to:
    • determine if a data set is modeled by a normal distribution.

    • compute how likely it is for a random variable underlying the data set to be normally distributed.

    • Testing for Normality is often used when analyzing the difference between groups using parametric tests.

    • A common assumption is that the dependent variable is approximately normally distributed for each group of the independent variable.

    Normal Distribution

  • A normal distribution describes a distribution of data that resembles a bell shaped symmetrical curve, where half of the data falls before and after the mean.

  • In a normal distribution, the mean, median and mode are generally equal.

  • The total area under the curve is 1.

  • A normal distribution is part of probability theory, which is a type of continuous probability distribution for a real-value random variable.

  • The general form of its probability density function: \[f(x) = {\frac{1}{\sigma \sqrt{2 \pi}}e^{-\frac{1}{2}(\frac{x - \mu}{\sigma})^{2}}}\]
    • where \(\mu \) is the mean.
    • where \(\sigma \) is the standard deviation.
    • where \(\sigma^{2} \) is the variance of the distribution.

  • A percentage of the data falls within a certain number of standard deviations from the mean, as illustrated in Standard Normal Distribution Curve.

  • Besides the percentages, many scores are derived from the normal distribution, such as percentile ranks, normal curve equivalents, and most commonly, t-scores and z-scores.

    Numeric and Graphical Methods of assessment for Normality

    Numerical methods Graphical methods
    Advantage: allows one to make an objective judgement of normality. Advantage: allows one to make a judgement to assess whether the data is normaly distributed.
    Disadvantage: May not detect violation of normality in small sample sizes. Disadvantage: Inspecting graphs/plots comes at the expense of objectivity.
    Disadvantage: Overly sensitive to large sample sizes, as small deviations from normality are detected. Methods include (but not limited to): Histographs, Normal Q-Q plot.
    Disadvantage: This method alone doesn't let you explore the data as graphical methods would.
    Methods include (but not limited to): Skewness/Kurtosis, and Shapiro-Wilk Test.

    Testing for Normality Method Selection

  • Generally, several methods are used concurrently to evaluate normality.

  • For those will little statistics background, graphical methods are the easiest to interpret.

  • Numeric methods are best when the sample size is small.

  • This post will focus on the graphical plot called a Normal Q-Q Plot.

    Normal Q-Q Plot

  • Also known as a Quantile-Quantile Plot.

  • Is a graphical tool used to assess if a data set came from a normal or exponential distribution.

  • Normal Q-Q plots is a visual check that allows users to see at-a-glance how the data may deviate from a theoretical normal distribution.

  • The Q-Q plot plots every observed value against a standard normal distribution with the same number of points.
  • Anatomy of a Q-Q plot
    • A Q-Q plot consists of two elements, a scatter plot and a straight solid line.

    • The scatter plot usually represents the data we are testing.

    • The solid line represents the standard normal distribution with the same number of datapoint as the data set of interest.

    • There are two methods for plotting Q-Q plots:
      1. Observed versus Expected
      2. Normal Theoretical Quantiles versus Normal Sample Data Quantiles
    • Observed versus Expected:
    • The Observed versus Expected places the Observed Values on the x-axis, and the Expected Normal Values on the y-axis.

    • The domain of the x-axis (if not normalized), is the range of the data set.

    • The range of the y-axis (if not normalized) consists of the standard normal distribution, and is between [-3, 3].

    • The observed data is sorted in ascending order and plotted along quantiles.

    • Quantiles are point in a data set that determine how many values are above, or below.

    • An example of quantiles are depicted in the Standard Normal Distribution Curve, where the 50 percent of the data (the 0.5 quantile) is to the left of the mean (half the data lie below the 0.
    • Normal Theoretical Quantiles versus Normal Sample Data Quantiles:
    • The Normal Theoretical Quantiles are plotted along the x-axis, and the Normal Sample Data Quantiles are plotted on the y-axis.

    • The domain of the x-axis is between [-3, 3].

    • The range of the y-axis is the range of the data set we are testing for normality.

    • The observed data is sorted in ascending order and plotted along quantiles.

    Interpretation Normal Q-Q Plot

  • Interpreting a Normal Q-Q Plot involves looking at how the scatter plot lines up along three parts of the solid line

  • If the scatter plot lines up with the straight vertical solid line in the center, and along both ends of the line, then there is evidence to suggest the data come from a normal distribution.

  • If the scatter plot only lines up in the middle of the solid line, but then deviates on both ends of the line, then it suggest the data is too peaked in the middle.

  • If the scatter plot is convex or concave, then it suggest the data is skewed and asymmetrical.

  • If the scatter plot has an "s" shape, then the data has positive kurtosis.

  • If the scatter plot has a thunderbolt shape, then the data has negative kurtosis.
