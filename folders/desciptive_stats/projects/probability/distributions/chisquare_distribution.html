Primer on Statistical Distributions


    What are statistical distributions?
  • A statistical distribution is the distribution that describes the relative number of times each possible outcome will occur in a number of trials.

  • Statistical distributions provide a parameterized mathematical function that can be used to calculate the probability of any instance or observation from a sample data.

  • Statistical distributions are described by probability distributions functions, and two commonly probability functions is the Probability density function (PDF) and the Cumulative distribution function (CDF).

  • The Probability density function (PDF) gives the probability of a given value will occur.

  • The Cumulative distribution function (CDF) describes the cumulative probability that a given value or any value smaller than it will occur.

    How are distributions mathematically defined?
  • Formally, a distribution is defined as a normalized measure, with the distribution of a random variable x is the measure \(P_{x} \) on \(\mathbb{S}' \) defined by setting:

    \[P_{x} (A')= P\{s\in S: x(s) \in A' \} \newline \text{where (S,}\ \mathbb{S} \text{, P) is a probability space} \newline \text{(S,}\ \mathbb{S}\ \text{) is a measurable space} \newline \text{and}\ P\ \text{a measure on}\ \mathbb{S}\ \text{with } P(S)=1 \]

    Why are distributions important?
  • Probability distribution functions are a fundamental concept in statistics and machine learning.

  • They are used to calculate confidence intervals for parameters and the critical regions for hypothesis testing.

  • Distributions are useful to determine a reasonable distribution model for univariate data (data which consists of observations on only a single characteristic or attribute).

  • Statistical intervals and hypothesis test are based on specific distributions, and knowing which distribution best models the data can help verify that the selected test is justified.

  • Probability distributions help us build model from our data, enabling us to obtain estimates of the probability that a certain event may occur, or estimate the variability of occurrence.

    Features of data that help characterize its distribution.
  • Before an analysis is performed on a given data set, it is important to characterize it by asking the following questions:
    1. The data take on discrete or continuous values. Discrete data takes are numeric whole, concrete numbers with specific and fixed values. Continuous data includes complex numbers of varying data values.
    2. When the data is plotted, what features of the line describe the symmetry. If it is asymmetric, which direction is does it lie.
    3. Are there upper and lower limits on the data. Example, data does not go below zero, or does not exceed a specific value.
    4. What is the likelihood of observing extreme values in the distribution.

  • Probability distributions fall into two major categories, those that are discrete or continuous.

  • Continuous probability distribution functions represent many commonly used models in statistics and machine learning, such as the Gaussian, chi-square and t distribution.

  • The following sections will go into the mathematical intuition for these two categories, and the subsequent sections will survey major distributions in these two categories.

    Discrete Distributions
  • Discrete distributions have values that take on numeric whole, concrete numbers with specific fixed values.

  • The mathematical definition of a discrete probability function p(x) has the following properties:
    1. The probability of x can take on specific values:
      \[P[X = x]\ =\ p(x)\ =\ p_{x} \]

    2. p(x) is non-negative for all real x values.

    3. The sum of p(x) over all possible values of x is 1.
      \[\Sigma_{j}p_{j} = 1 \]
      where j is all possible values that x can have
      and \(p_{j} \) is the probability at \(x_{j} \)


  • The result of properties 2 and 3 is that all probabilities of x are between [0, 1].

  • A discrete probability function can take a discrete number of values.

  • In practice, discrete probability functions only take on integer values.

  • Each discrete value has a certain probability of occurring that is between 0 and 1.

  • A discrete function that allows negative values or values greater than one are not a probability function.

  • Finally, the sum of all probabilities is one, means that at least one of the values has to occur.

  • Discrete probability functions are referred to as probability mass functions

    Continuous Distributions
  • Continuous data includes complex numbers of varying data values.

  • The mathematical definition of a Continuous probability function f(x) has the following properties:
    1. The probability that x is between point a and b is:

      \[p[a \leq x \leq b] = \int_{a}^{b} f(x)dx \]

    2. Values are non-negative for all real x.

    3. The integral of the probability function is one:

      \[\int_{\infty}^{-\infty} f(x)dx = 1 \]


  • Since a continuous probability functions is defined over an infinite number of points over a continuous interval, the probability of a single point is always zero.

  • Probabilities are measured over intervals and not single points, and the area under the curve between these two distinct points define the probability for that interval.

  • The height of the probability function can be greater than 1.

  • The integral for continuous probability functions must be equal and is equivalent to the property for discrete distributions.

  • Continuous probability functions are referred to as probability density functions


Chi-Square Distribution


    The Basics
  • Chi-Square distribution is a continuous probability distribution with k degrees of freedom..

  • Chi-Square distribution is one of the most commonly used discrete distribution.

  • Chi-Square distribution is the distribution of a sum of the squares of the k independent standard normal random variables.

  • Chi-Square distribution is a special case of the gamma distribution, and is one of the most widely used probability distributions in inferential statistics.

  • Two common statistical tests that relies on the Chi-Squared distribution are the Chi-square goodness of fit test and the Chi-square test of independence.

  • The distribution is defined by the degrees of freedom, and will change the distribution, as the number of degrees of freedom increases, the distribution takes on a normal curve.

  • Since the Chi-square distribution can be used wherever a normal distribution can be used for a hypothesis test.

  • The Chi-square distribution is also used in these hypothesis tests
      Examples
    • Chi-square test of independence in contingency tables
    • Chi-square test of goodness of fit of observed data to hypothetical distributions
    • Likelihood-ratio test for nested models
    • Log-rank test in survival analysis
    • Cochran–Mantel–Haenszel test for stratified contingency tables

  • The Chi-square test is also a component of the definition of the t-distribution and the F-distribution used in t-tests, analysis of variance, and regression analysis.

  • The Chi Square distribution is very important because many test statistics are approximately distributed as Chi Square.


Mathematical Intuition for Chi-Square Distribution


    Chi-square basics
  • If \(Z_{1}...Z_{k}\) are independent, standard normal variables, then the sum of their squares would be: \[Q = \Sigma_{i=1}^{k}Z_{i}^{2}\]

  • Q is distributed according to the chi-square distribution with k degrees of freedom and is denoted as:
    \(Q ~ \chi^{2}(k)\) or \(Q ~ \chi^{2}_{k}\)

  • Note that the chi-square distribution only has one parameter: a positive integer k that specifies the number of degrees of freedom.

  • The degrees of freedom are the number of random variables that are summed up \(Z_{i}\).

    Mathematical Intuition for Chi-Square Distribution
  • \[X = \frac{m - N_{p}}{\sqrt{Npq}} \] where m is the number of observed successes in N trials and the probability of success is p. Therefore q = 1 - p.

  • Squaring both sides of the equation gives us: \[X^{2} = \frac{(m - N_{p})^{2}}{Npq} \]

  • if we then use the following formulas \[N = N_{p} + N(1 - p)\], \[N = m + (N - m)\], and q = 1 - p

  • \[X^{2} = \frac{(m - N_{p})^{2}}{Np} + \frac{(N - m - N_{q})^{2}}{Nq} \]

  • This is then generalized to the following form \[X^{2} = \Sigma_{i=1}^{n}\frac{(0_{i} - E_{i})^{2}}{E_{i}}\]

  • \(X^{2} \) is Pearson's cumulative test statistic
  • \(O_{i} \) is the number of observation of type i
  • \(E_{i} = N_{pi} \) is the expected frequency of type i asserted by the null hypothesis that the fraction of type i in the population is \(p_{i} \)
  • n is the number of cells in the table.
    Probability Density Function
  • \[f(x) = \frac{e^{\frac{-x}{2}}x^{\frac{v}{2}-1}}{2^{\frac{v}{2}}\Gamma(\frac{v}{2})}\] for \(x \geq 0 \)
  • v is the shape parameter \(\Gamma\) is the gamma function

  • The gamma function is then described as \[\Gamma(a) = \int_{0}^{\infty}t^{a-1}e^{-t}dt \]

  • The chi-square distribution is treated as a standard distribution with no location or scale parameter, but it can be transformed with a location parameter \(\mu\) and a scale parameter \(\Sigma\).

    Cumulative Distribution Function
  • \[f(x) = \frac{\gamma(\frac{v}{2}, \frac{x}{2})}{\Gamma(\frac{v}{2})} \] for \(x \geq 0 \)
  • where \(\Gamma\) is the gamma function described in the previous example.

  • and \(\gamma\) is the incomplete gamma function that is described as \[\gamma_{x}(a) = \int_{0}^{\infty}t^{a-1}e^{-t}dt \]


Common Binomial Distribution Statistics


  • Mean: \(v\)

  • Median: approximately \(v - \frac{2}{3}\) for large v

  • Mode: \( v - 2\) for \(v > 2 \)

  • Range: \(0\ to\ \infty\)

  • Standard Deviation: \(\sqrt{2v}\)

  • Coefficient of Variation: \(\sqrt{\frac{(2)}{v}}\)

  • Skewness: \(\frac{(2^{1.5}}{\sqrt{v}} \)

  • Kurtosis: \(3 - \frac{12}{v}\)

    Parameter Estimation
  • Chi-square distribution is typically used to develop hypothesis tests and confidence intervals and rarely for modeling applications

  • The chi-square distribution is used in many cases for the critical regions for hypothesis tests and in determining confidence intervals.


Visualizing the Chi-Square Probability Density Function


    Why we plot the Probability Density Function (PDF)
  • The Probability density function gives the probability of a given value will occur.

  • Visualizing PDF gives a graphically portrayed off a given probability and the area under the curve will indicate the interval in which the variable will fall.

  • The probability density plot is used to understand the data distribution for a given variable and we want to know the likelihood (or probability) of obtaining a range of values that the variable can assume.

    How to plot PDF using python scipy (v1.6.3) and matplotlib.
  • The scipy \(\chi^{2} \) class provides a chi-squared continous random variable.

  • scipy.stats.chi2 in this package is defined as:
    \[f(x,k) = \frac{1}{2^{\frac{k}{2}}\Gamma(\frac{k}{2})}x^{\frac{}k{2-1}}exp(\frac{-x}{2} \]
    \[for\ x\ > 0\ and\ k > 0\]
    \[and\ k = degrees of freedom\]

  • This class has several methods that can be used on a A chi-squared continuous.

  • To plot the probability mass function, we will use the percent point function chi2.ppf(q, df, loc=0, scale=1) and the probability density function chi2.pdf(x, df, loc=0, scale=1)

  • chi2.ppf(q, df, loc=0, scale=1): takes the quartile and the degrees of freedom. We obtained the ppf for the 1% and the 99% percentiles and then used the numpy method numpy.linspace to return an array of 100 data points between the 1% and 99% and saves all this data into the x variable.

  • chi2.pdf(x, df): uses the array of 100 values between 0.1 and 0.99 of a and returns the chi-square probability distribution.

  • The df (or k) will affect the overall shape of the curve, and as the df (k) increases, the curve begins to resemble a normal distribution.

  • In example below, we will generate four subplots, each subplot will have a different value for p
  • Note that as the probability for success increases, the binomial probability mass function will shift to the right, and vice versa.

degrees_of_freedom = [1, 3, 6, 10] # the degrees of freedom
fig = plt.figure(figsize=(12,8))
for df, num in zip(degrees_of_freedom, range(1,5)):
    
    ax = fig.add_subplot(2,2, num)
    x = np.linspace(chi2.ppf(0.01, df), chi2.ppf(0.99, df), 100)
    ax.plot(x, chi2.pdf(x, df), 'r-', lw=5, alpha=0.6, label='chi2 pdf')
    rv = chi2(df)
    vals = chi2.ppf([0.001, 0.5, 0.999], df)
    r = chi2.rvs(df, size=1000)
    ax.hist(r, density=True, histtype='stepfilled', alpha=0.1)
    ax.legend(loc='best', frameon=False)
    ax.set_ylabel('Probability Mass', fontsize=14)
    ax.set_xlabel('x', fontsize=14)
    ax.set_title(f"Chi-Square PDF (k={df}, n={len(x)})")

plt.tight_layout()
plt.show()


Visualizing the Chi-Square Cumulative Density Function


    Why we plot the Chi-Square Cumulative Density Function
  • The Cumulative distribution function (CDF) describes the cumulative probability that a given value or any value smaller than it will occur.

  • The chi-square cumulative distribution function lets one obtain the probability of observing less than or equal to x successes in n trials, with the probability p of success on a single trial.

    The mathematics of CDF
  • \[f(x) = \frac{\gamma(\frac{v}{2}, \frac{x}{2})}{\Gamma(\frac{v}{2})} \] for \(x \geq 0 \)
  • where \(\Gamma\) is the gamma function described in the previous example.

  • and \(\gamma\) is the incomplete gamma function that is described as \[\gamma_{x}(a) = \int_{0}^{\infty}t^{a-1}e^{-t}dt \]

    How to plot CDF using python scipy (v1.6.3) and matplotlib.

  • To plot the cumulative distribution function, we will use cdf(x, df)
