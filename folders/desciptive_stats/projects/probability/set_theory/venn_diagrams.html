Introduction to Set Theory.


    The Basics
  • Set theory is a branch of mathematical logic that studies collections of objects (formally called sets.

  • A Set is an unordered collection of non-repeating objects.

  • Objects in sets are also referred to as elements or members.

  • Georg Cantor is regarded as the father of modern set theory, and this branch of mathematics was founded by his 1874 paper "On a property of the collection of All Real Algebraic Numbers".

  • Venn diagrams are used to illustrate basic concepts and binary operations on sets.

    Applications
  • Set theory is used to define graphs, manifolds, rings and vector spaces.

  • Set theory can be used to derive properties of natural and real numbers.

  • Set theory serves as a foundation for mathematical analysis, abstract algebra, probability, and discrete mathematics.

    Describing a set
  • A set is described by listing all the elements inside a curly bracket or described using a written description inside curly brackets.

  • \(S = \{1,3,5,7,9\} \).

  • S is the set whose elements are 1,3,5,7,9.

  • General form is \[A = \{x : property\} \].

    Set Notation
  • \(\varnothing \) Empty Set

  • \(\mathbb{N} \) Set of Natural Numbers

  • \(\mathbb{Z} \) Set of Integers

  • \(\mathbb{Q} \) Set of Rational Numbers

  • \(\mathbb{A} \) Set of Algebraic Numbers

  • \(\mathbb{R} \) Set of Real Numbers

  • \(\mathbb{C} \) Set of Complex Numbers

  • \(\in \) is member of

  • \(\notin \) is not a member of

  • \(\ni \) owns (has member)

  • \(\subset \) is proper subset of

  • \(\subseteq \) is subset of

  • \(\supset \) is proper superset of

  • \(\cup \) set union

  • \(\cap \) set intersection

    Types of Sets
  • Empty Set does not contain any elements, and is denoted as \(\varnothing \) or {}

  • Singleton Set contains a single element.

  • Finite Set contains a definite number of elements.

  • Infinite Set is not finite, such as a set of all natural numbers \(\mathbb{N} \).

  • Equivalent Set describes a situation when the number of elements is the same for two different sets. The order of sets does not matter. \(n(A) = n(B) \) where A and b are two different sets

  • Equal Set the two sets A and b are equal if they have the exact same elements. \(S_{A} = \{1,2,3,4\} \) = \(S_{B} = \{1,2,3,4\} \), thus \(S_{A} = S_{B} \)

  • Disjoint Set Condition in which two sets that are compared do not contain any common elements.
    \(S_{A} = \{1,2,3,4\} \) = \(S_{B} = \{5,6,7,8\} \), thus \(S_{A} \neq S_{B} \)

  • Subset A set is a subset if every element is also an element in another set.
    \(S_{A} = \{1,2,3,4\} \) = \(S_{B} = \{1,2,3,4,5,6,7,8\} \), thus \(S_{A} \subseteq S_{B} \)
    a set is also a subset of itself. if a set is not a subset of another set, such as \(S_{C} = \{1,2\} \) = \(S_{D} = \{6,7,8\} \), , then it would described as \(S_{C} \nsubseteq S_{D} \)

  • Proper Subset is a set that meets two conditions: 1) all the elements are contained in a larger set \(S_{A} = \{1,2\} \in S_{B} = \{1,2,3,4\} \) and 2) the proper subset is not an equal set to the larger set \(S_{A} \{1,2\} \neq S_{B} \{1,2,3,4\} \)

  • Super Set A set is a superset of another set if all elements in one set are represented in the superset.
    \(S_{A} = \{A,B,C,D\} \supset S_{B} = \{A,C\} \)

  • Universal Set is a set that contains all the sets relevant to a certain condition.
    \(S_{Universal} = \{A,B,C,D\}, S_{A} = \{A,C\}, S_{B} = \{B,D\} \)


Set Operations and Formulas.


    Common Operations
  • Union of Sets: Is the set that contains all the elements of two sets and is denoted using the \(\cup\) symbol, example: \(A \cup B\)

  • Intersection of Sets: Is the set that contains only common elements between sets and is denoted using the \(\cap\) symbol, example: \(A \cap B\)

  • Complement of Sets: Is the set of all elements in the universal set that are not in a specific set and is denoted using an apostrophy example: Set \(P'\) has elements that are not in \(P\)

  • Difference of Sets: if set A and set B are two sets, then the difference of sets for A as denoted as \(A - B\) is the set that contains elements in A but not in B.

    Common Set Formulas with Probability
  • Additive Law: \(n(A \cup B\) = \(n(A) + b(B) - n(A\cup B)\)

  • Complement Rule: \(n(A') = 1 - n(A)\)

  • Law of Total Probability: \(P(B) = P(A \cap B) + P(A' \cap B) \)


Introduction to Venn Diagrams.


    The Basics
  • A venn diagram is a diagram used to show logical relationships between sets.

  • Venn diagrams are also used to illustrate relationships in probability, logic, statistics, linguistics and computer science.

  • Venn diagrams typically use simple shapes (ellipses, circules, and squares) to represent set space.

  • Venn diagrams are also called primary diagrams, set diagrams, and logic diagrams.

  • Venn diagrams depict elements as points in a plane, and the set as regions inside closed shapes. Venn diagrams may consists of multiple

  • In the example below, we have 40 elements in a sample space.

  • In this sample space, we have two sets represented by red and green.

  • The orange segment between Set A and Set B are the elements that Set A and Set B have in common, denoted as \(A \cap B\)

  • The top left of the venn diagram we have a value of 6, which indicates the number of elements in this sample space that is not in either Set A or Set B, denoted as \((A \cup B)' = 1 - (A \cup B)\)

  • From this diagram, we can easily calculate the probability of a given occurrence of an event as

    \[\frac{Occurrence}{All events} \]

Use Venn Diagrams to Illustrate the Law of Total Probability.


    The basics
  • The total law of probability is a fundamental rule that expresses the total probability of an outcome.

  • The theorem states that in a discrete case, if \[\{B_{n}: n = 1, 2, 3...\} \]
    is a finite of a sample space, and each event in \(B_{n}\) is measurable, than for an event \(A\), of the same probability space:

    \[P(A) = \Sigma_{n}P(A\cap B_{n}) \]

  • This expression can also be written as: \[P(A) = \Sigma_{n}P(A | B_{n})P(B_{n}) \]
    where, for any \(n\) for which \(P(B_{n}) = 0\) these terms are omitted from the summation, between \(P(A|B_{n}) \) is finite

  • The law of probability can also be stated for conditional probabilities as:

    \[P(A|C) = \Sigma_{n}P(A | C\cap B_{n})P(B_{n} |C) \]

    taking the \(B_{n}\) as above and assuming C is an event independent of any of the \(B_{n}\)

    \[P(A|C) = \Sigma_{n}P(A | C\cap B_{n})P(B_{n}) \]


  • Finally, this expression can also be written as: \[P(B) = P(A \cap B) + P(A' \cap B)\]

  • In the example below:
    \[P(A \cap B) = 12/52\]
    \[P(A'\cap B) = 4/52\]
    \[P(B) = 12/52 + 4/52 = 16/52 = 0.31\]

<h1 style="text-align: center;" }>The Law of Total Probability using Venn Diagrams

[P(B) = P(A \cap B) + P(A' \cap B)]</h1>

Use Venn Diagrams to discuss the additive law.


    The Basics
  • States that the probability of two events is the sum of the probabilities that either will happen minus the probability that both will happen.

  • Note that the last term has been accounted for twice in \(P(A)\) and in \(P(B)\), so it must be subtracted once so that it is not double counted.

  • If A and B are disjointed, then \(P(A \cap B) = 0\) and the formula becomes \(P(A \cup B) = P(A) + P(B)\).

  • In the example below:
    \[P(A) = 42/52\]
    \[P(B) = 16/52\]
    \[P(A \cup B) = 12/52\]
    \[P(A \cup B) = 42/52 + 16/52 - 12/52 = 46/52 = 0.88\]

<h1 style="text-align: center;" }>Additive Law

[P(A \cup B) = P(A) + P(B) - P(A \cup B)]</h1>

Use Venn Diagrams to Complement Rule.


    The Basics
  • States that the sum of the probabilities of an event and its complement must be equal to 1 and is expressed as \[P(A) + P(A') = 1 \].

  • It is sometimes easier to calculate the probability of the complement of an event than it is to calculate the probability of the event itself, and this is done by rewriting the expression as \[P(A') = 1 - P(A) \]

  • In the example below:
    \[P(A) = 42/52\]
    \[4252 + P(A') = 1\]
    \[P(A') = 1 - 42/52\]
    \[P(A') = 0.19\]

<h1 style="text-align: center;" }>Complement Rule

[P(A') = 1 - P(A)]</h1>
