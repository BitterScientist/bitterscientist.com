Background Info

Brendan Eich, 2012

Javascript: is one of the core world wide web technologies that was created in the 1990s by Netscape communications corporation and Brendan Eich. Javascript, also known as JS, is a high-level, interpreted language that is used to enable interactive web pages, and power many web applications. In fact, most web browsers have dedicated javascript engines to execute these commands.

Why is this important?
Javascript has been, and continues to be a ubiquitous online element, and is critical to learn for those interested in web based technologies.
