Jupyter Notebook

Background Info

Fernando Perez, 2012

Jupyter Notebook: is an open source software developed to execute environments in several dozen languages, and is a spin off from iPython in 2014 by Fernando Perez. Jupyter Notebook is commonly used for interactive data science and scientific computing. Project Jupyter

Why is this important?
Jupyter notebook makes it easy for users to:

  • carry out data-driven experiments in an interactive computation environment
  • gives users better version control over their code blocks
  • allows users to share code, reproduce results, and encourage collaboration
