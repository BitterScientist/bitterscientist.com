Export Jupyter Notebook to HTML

Updated: 10-30-2019

Background Info

Blogging with Jupyter Notebook: although Jupyter Notebooks are by designed, sharable, you sometimes wants to display your notebook as a pdf or an html page. To do this without having to print your notebook to pdf, or screen shot your notebook for html embedding, use Jupter Notebook'sDownload as feature.

Why is this important?
Data science, like any science, is intended to be shared, and this tool enables you to do so using traditional means of dissemination. This is especially useful if you are a data science blogger like bitterscientist!

Export Jupyter Notebook to HTML

  • Step 1: Select File from toolbar
  • Step 2: From dropdown menu, select "Download As"
  • Step 3: From dropdown menu, select "HTML (.html)"
