Convert Jupyter Notebook into HTML using nbconvert

Background Info

Blogging with Jupyter Notebook: although Jupyter Notebooks are by designed, sharable, you sometimes wants to display your notebook as a pdf or an html page. To do this without having to print your notebook to pdf, or screen shot your notebook for html embedding, use nbconvert

nvconvert: is an open source software that enables you to:

  • Publish using LaTex
  • Opens the door for embedding notebooks in papers
  • Share and collaborate with others who do not use jupyter notebook

Why is this important?
Data science, like any science, is intended to be shared, and this tool enables you to do so using traditional means of dissemination. This is especially useful if you are a data science blogger like bitterscientist!

Step 1: Install nbconvert

  • Open Mac Terminal
  • pip install nbconvert
  • Additional Installation Instructions: nbconvert 6.0.7

Step 2: Run nbconvert from the Command Line

  • General usage code: jupyter nbconvert --to <output format> <input notebook> This method will include all html and css elements
  • Where <output format> is the desired file output
  • Where <input notebook> is the is the filename of the Jupyter notebook. Note: make sure that you either include the filepath, or that you are running this code in the file directory
  • Example jupyter nbconvert --to html markdown-tables.ipynb
  • The html file will be saved in the same directory as your input notebook
  • To output a basic HTML for embedding into another HTML page, use this Optional Argument --template basic
  • Example jupyter nbconvert --to html markdown-tables.ipynb --template basic this method will yield HTML without the head, and body elements of a page.
  • Additional Installation Instructions: nbconvert 5.6.1

Step 3: Display converted jupyter notebook in webpage

  • If you used: jupyter nbconvert --to <output format> <input notebook> then the html is ready for use.
  • If you used: jupyter nbconvert --to <output format> <input notebook> --template classic then you need to embed it in an html page.
