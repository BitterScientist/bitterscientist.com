Assumptions.


    The Basics:
  • The assumptions for the ANOVA include:
    • the k samples must be drawn randomly from k distict populations

    • the k samples are selected independnetly of one another

    • the response variable (dependent) has a normal distribution for each of the k populations

    • The standard deviation for the dependent variable (response variable) has the same population standard deviation $\sigma$ among the k populations (homoscedasticity).


  • The residuals are the difference between the observed and predicted values of data.

  • Confirming these assumptions is essential before performing an ANOVA, and can be accomplished by performing a Shapiro-Wilk test for normality or visually using a residual plot or q-qplots.

  • The following R code will demonstrate how to perform these procedures using data for the protein content in several types of grain.

Assessing Homoscedasticity¶

  • performing a homogeneity of variance hypothesis test
  • Using a graphica tool

Tests for Homogeneity of Variance¶

  • Uses an F-test to determine if the two populations have the same variance
  • The hypothesis test is $$H_0\colon\sigma_1^2=\sigma_2^2=\dotsb=\sigma_k^2\quad\text{against}\quad H_a\colon\text{there exists some difference in the $\sigma_i^2$'s}$$
  • The Leven's test is a popular procedure
  • A One-Way ANOVA procedure on the squares of residual is another method, as illustrated below using the anova(aov(response_variable.residual^2~independent_variable))
  • In the example below, the p-value is over 0.05, which means that we retain the $H_0$, and the groups have equal variance (homoscedasticity)
In [4]:
anova( aov( grain.res^2~grain ) )

Plots to visualize residuals against fitted values¶

  • This method places the One-Way ANOVA in the context of the general linear model
  • The method evaluates the residuals (the observed error terms) from the model
  • The normal q-q plot on the set of residuals asses the assumption that the error term is normally distributed
  • To assess homoscedasticity, we then look at the scatter of residuals against the fitted values from the model
  • In a one-way ANOVA the k fitted values $\hat{\mu_{i}} = \bar{y_{i}}$, for each of the k group means.
  • we can apply this method in R as follows
  • You want to see roughly equal ranges for each x-value.

Plot student residuals against the fitted data¶

  • The studentized residuals is just the residuals divided by the root mean square
  • The root mean square is the the standard deviation of the sample residual
  • We create this plot by calculating the root.MSE and dividing the residuals by this value
  • We then plot the fitted values by this series
  • We want to see that the residuals have roughtly the same range for each x value.
In [7]:
root.MSE <- 0.869962
grain.stu_res <- grain.res/root.MSE # create studentized residuals by dividing by root.MSE
plot(x=grain.fit, y=grain.stu_res)


Violations Assumptions..


    The Basics:
  • If after performing this procedures there and the assumptions of normality and homoscedasticity are violated, then you will need to consider alternative approaches.

  • Wilcoxon signed rank test: use this as a non parametric alternative to the paired t-test

  • Wilcoxon-Mann-Whitney rank sum test: use this as a nonparameric alternative to the two-independent-samples t-test

  • Kruskal-Wallis test: use this as an alternative to the ANOVA but it has the following assumptions:
    • k random samples of response variables that are either numeric or ordinal

    • k samples are independent of one another

    • the distribution of the responses in each population have the similar shapes when plotted.



  • The hypothesis test for the Kruskal-Wallis test is: $$\begin{aligned} &H_0\colon\text{The $k$ populations have the same distribution}\\ \text{against }&H_a\colon\text{At least one of the $k$ populations are "shifted" from the others}\end{aligned}$$

  • The example below will illustrate how to perform this in R using a sample dataset of oil spills.
