Overview of Assumptions in Linear Regression.


    The Basics:
  • Many of the assumptions are concerned with the $\epsilon_j$ term which cannot be directly observed

  • The linear model is appropriate for the response and explantory variables.

  • The error term $\epsilon_j$ is independent from one another

  • The error term is drawn from a normal distribution

  • The mean of the error term $\epsilon = 0$, which will always be met using the least-square regression,

  • The least-square regression estimates of $\hat{y_j}$ will gurantee that the sum of the residuals is 0, which makes the mean of the error term $\epsilon = 0$

  • The variance of all error terms is $\sigma^2$

  • Since we can't observe the error, we can use the residuals as a proxy for the $\epsilon$

    Residual Plots:
  • Since the residuals are important at understanding the error of the model, there are several plots that we can use to evaluate the residuals in the model.

  • The normal q-q plot of the sample residual is often used to test the assumption of normality. The larger the sample size the less concerned we can be abotu the non-normal population of error terms.

  • The plot of the residuals against the fitted values is useful in assessing homoscedasticity (equal variance of the error terms)

  • The plot of the residuals against all the explanatory variables will give us the esame conclusion as the other residual plots when applied to a simple linear regression. But it is useful in other contexts.

    Diagnosing Issues with Residual Plots:
  • Homoscedasticity issues with Fitted vs Residual Plot:
      A fan-shape
    • majority of points are in the middle and fan out at either ends

    • majority of points are at either ends of the fitted axis and minimal in the center of the fitted axis

    • points fan out from the lower end of the fitted axis

    • points fan out from the upper end of the fitted axis


  • The MASS Rubber dataset will be used to illustrate some of these patterns and methods in R that can be used to plot the residuals.

Perform Linear Regression Model and plot Boxplot, Histogran and Scatter plot of Residual and Fitted Values¶

  • residual() # to get residuals
  • fitted.values() # to get fitted values
  • The points on the normal q-q plot do not evenly line up to the diagonal line, but the box plot and histogram of the residuals suggest data is normally distributed.
  • The residual to fitted values plot also suggest the residuals are evenly distributed accross all points of the fitted values, which suggest homoscedasticity.
In [2]:
loss.lm <- lm(loss~hard)
loss.res <- residuals(loss.lm)
loss.fit <- fitted.values(loss.lm)

# plot qqnorm and qq line
qqnorm(loss.res)
qqline(loss.res)

# plot residual versus fitted
plot(loss.res~loss.fit)
abline(h=0)

# plot boxplot
boxplot(loss.res)

# plot histogram of residual
hist(loss.res)


Transformation.


    The Basics:
  • If the plot of residuals against the fitted values violates homoscedasticity or an appropriate linear model, then a option is to transform the data.

  • Transformation is a method that rescales the data so that makes the data more linearly related.

    • Guidelines for transformation data with a strong right skew and residual plots violate assumption of fitted model.
    • For the independent variable try fitting the model with $ln()$ or $\sqrt{x}$

    • For the response (dependent) variable try fitting the model with $$\ln(y)=\beta_0+\beta_1\ln(x)+\varepsilon$$


  • If the plot of the residuals against the fitted values has a fan-shape, try applying a natural log transformation on the response variable y.

  • The following example will use the MASS forbes dataset, which relates boiling water to air pressure.

Transform data using log to adjust the residual and fitted plot¶

# log transformation on both response and independent variable
forbes.lm2 <- lm( log(pres) ~ log(bp) ) 
plot( log(pres) ~ log(bp) ); abline(forbes.lm2)

Plot the residual vs fitted scatter plot and the normal qq plot¶

  • The log transformation has improved the fitted versus residual plot BUT there is still a crazy outlier.
  • The outlier can only be removed if there is clear evidence of an experimental error during the collection process
