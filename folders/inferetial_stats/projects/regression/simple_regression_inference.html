Introduction to Simple Linear Regression.


    The Basics:
  • A similar linear gression can be obtained from a set of paired data (x, y) using this formula $$\hat\mu_{y|x}=\hat\beta_0+\hat\beta_1x$$.


  • The issue though is determining if the point estimates (the parameters for the slope, intercept and error) are useful.

  • This post will discuss the inference test that can be performe on simple linear regression and determine if:

    $$H_0\colon y=\beta_0+\varepsilon\quad\text{against}\quad H_a\colon y=\beta_0+\beta_1x+\varepsilon$$

  • The null hypothesis suggest that x does not belong to the linear model, and we are looking for evidene to prove that it does, and we specifically test this hypohtesis with this expression:

    $$H_0\colon\beta_1=0\quad\text{against}\quad H_a\colon\beta_1\ne0$$.

  • To this this last hypothesis we would need:
    • A point estimate for $\beta_1$

    • The standard error of the point estimate $\widehat{\mathsf{SE}}_{\hat\beta_1}$

    • The degrees of freedom associated with the standard error.

    • Confirmation that the sampling distribution for the $\hat \beta_1$ is approximately normals



  • We can then use the t-statistic $$t=\frac{\hat\beta_1-\beta_1}{\widehat{\mathsf{SE}}_{\hat\beta_1}}$$

  • The post will use R to demonstrate how to perform the calculations needed to test for inference on a simple linear model.

  • The R library MASS will be used, specifically the dataset Rubber to demonstrate such calcuation. The rubber dataset compares the tire rubber loss (grams) bsed on the tensile and hardness properties.


Perform a t-test on the slope parameter.


    The Basics:
    • Hypothesis test:
    • $$H_0\colon\beta_1=0\quad\text{against}\quad H_a\colon\beta_1\ne0,$$


    • Hypothesis test values:
    • Parameter of interest. $\beta_1$

    • Point estimate. $$\hat\beta_1=\frac{SS_{xy}}{SS_{xx}}$$

    • Standard error under $H_0$: $$\widehat{\mathsf{SE}}=\frac{\hat\sigma}{\sqrt{SS_{xx}}}$$

    • Test statistic. $$t=\frac{\hat\beta_1-0}{\widehat{\mathsf{SE}}}$$

    • Null distribution: student's t distribution

    • Degrees of freedom: $$df=n-2$$

    • The residualstandard error, variance: $$\sigma=\sqrt(MSE)$$


    • Confidence intervals: You can perform a one-sided test but that isn't common, so it wont be presented here.

    • Calculated as $100(1- \alpha)$

    • Two-sides: $$\hat\beta_1\pm t_{df,\alpha/2}\cdot\widehat{\mathsf{SE}}_{\hat\beta_1}$$


  • The following code will demonstrate these principles and calculations using R programming.

Estimating and Interpreting Parameters.


    The Basics:
  • The next step is to calculate the point estimate for $\sigma^2$ using the following formula $$\hat\sigma^2=\frac{SSE}{n-2}\quad\text{where}\quad SSE=\sum_{j=1}^ne_j^2=\sum_{j=1}^n(y_j-\hat y_j)^2=\sum_{j=1}^n(y_j-\hat\beta_0-\hat\beta_1x_j)^2$$

  • The SSE is the error sum of squares or the residual sum of squares.

  • This formula is one way to estimate the standard error of $\hat{\beta_1}$ and perform a hypothesis test on the $\beta_1$

  • Another way to partition the SST total variability observed in the response Y is as follows: $$\begin{aligned} SSR&=\sum_{j=1}^n(\text{fitted}-\bar y)^2\\ &=\sum_{j=1}^n(\hat y_j-\bar y)^2\\ &=\sum_{j=1}^n(\hat\beta_0+\hat\beta_1x_j-\bar y)^2\\ &=\frac{SS_{xy}^2}{SS_{xx}}\end{aligned}$$

  • The SSR is similar to the SSB (between groups sum of squares)

  • The SSR is the regression sum of squares or model sum of squares and this is the amount of variability observed in the response y that can be explained by the presence of x in the linear model.

  • SST = SSR + SSE

  • The SST is the sum of square deviations of the observed response $y_j$ from the overall mean $\bar y$ and it has a degrees of freedom n - 1. This is the measurement of variability in the model in the resposne y when we do not have an explanatory variable in the model.

  • The SSR is the gression sum of squares and is the sum of square deviations of the fitted values of the response $$\hat y_j=\hat\beta_0+\hat\beta_1x_j$$ from the overall sample mean $\bar y$.s

  • The SSE is the error sum of squares and is the sum of square deviations of the observed response $y_j$ from the predicted (fitted) value of the response $\hats{y_j}$

  • The degrees of freedom for the total sst is $$df_T=df_E+df_R$$

  • The degrees of freedom for the SSR is 1

  • The degrees of freedom for the SST is n - 1

  • The degrees of freedom for the SSE is n - 2


Perform an F-test of Model Fit.


    The Basics:
  • The F-test for the simple linear model is similar to the F-test for the ANOVA table.

  • The source of variability for
      Regression
    • $df_R$ = 1

    • SSR

    • MSR = $$\frac{SSR}{df_R}$$

    • F = $$\frac{MSR}{MSE}$$


  • The source of variability for
      Error
    • $df_E$ = n - 2

    • SSE

    • MSe = $$\frac{SSE}{df_E}$$


  • The source of variability for
      Total
    • $df_E$ = n - 1

    • SST


  • The next section will demonstrate how to perform these calculations in R programming.

Perform Confidene and Prediction Intervals.


    The Basics:
  • The last two inferences we want to make with simple linear regression pertain to the confidence and prediction intervals.

  • The confidence interval acts on the mean of the resposne y to some value in the explanatory variable x $$\mu_{y|x}=\beta_0+\beta_1x$$

  • a prediction interval acts on an individual obseration of the response y given some value of x and has a larger interval associated with it.

  • The following section will demonstrate how to calculate these intervals and plot them.

Plot confidence and prediction intervals¶

In [18]:
x1 <- min(hard)

# range of x values in the observed sample
x2 <- max(hard) 

# Vector of 200 x values equally spaced between the min and max
x <- seq(x1, x2, l=200) 

# data frame with the given x values
new.tires <- data.frame( hard=x ) 

# Confidence and prediction intervals off of data
conf.int <- predict(loss.lm, new.tires, interval='confidence')
pred.int <- predict(loss.lm, new.tires, interval='prediction')

# create the scatter plot
plot(loss~hard) 

# add the regression line
abline(loss.lm) 

# Add confidence interval bounds
lines(x, conf.int[,2], col='blue') 
lines(x, conf.int[,3], col='blue') 

# Add prediction interval bounds
lines(x, pred.int[,2], col='red') 
lines(x, pred.int[,3], col='red') 

# add legend
legend('topright', lty=1, col=c('blue', 'red'), legend=c('95% Confidence band', '95% Prediction band'))
