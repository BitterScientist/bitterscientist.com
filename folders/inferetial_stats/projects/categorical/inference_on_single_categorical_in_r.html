Introduction to Single Variable Categorical Inference.


    The Basics:
  • Categorical inferences are common in statistics, and includes single population propotion $p$, or the difference between two population proportions $p_1 - p_2$, which used binomial categorical data of success or failure.

  • Data can be coerced into taking two values, as if done with the sign test, which makes inferences on the population median.

  • We can also coerce data into two values when we classify households above or below the federal poverty line, or when we stratefy individuals as either blue eyed or brown eyed.

  • All these conditions are dealing with binomial data, and these methods are useful but restrictive to only two levels.

  • This post will consider multinomial data (categorical data with more than two levels) and use R programming to perform inferences and this datatype.


Goodness-of-Fit Test.


    The Basics:
  • The goodness-of-fit test is a large-sample hypothesis test that falls under the umbrella of the Pearson $\chi^2$ test.

  • This means that these test are always upper-tailed.

  • In R programming, we use the method pchisq(chi.statistic, df, lower.tail=FALSE) to find the p-value associated with the $\chi^2$ test statistic.

    Assumptions:
  • We have a single categocial variable with levels (subgroups) lebels as $1, 2, 3 .. k$

  • The categorical variable will have k levels (groups).

  • The population will have population proporitions for each level and is denoted as $p_1, p_2, p_3 .. p_k$

  • The sum of $p_1, p_2, p_3 .. p_k = 1$

  • In this test, we are seeking evidence that the population proportion deviates from the population proportions, as expressed with: $$\begin{aligned}&H_0\colon p_1=(p_1)_0,p_2=(p_2)_0,\dotsb, p_k=(p_k)_0\\\text{against }&H_a\colon\text{some deviation from the null proportions exists}\end{aligned}$$

  • Where the $(p_j)_0$ expression in the hypothesis is a population proportion.

  • The observed sample is randomly selected from the population.

  • The sample size is large enough to invoke the large sample approximation, whereby we have at least five sampling units within each category.

    Testing Notation
  • The hypothesis is tested by sampling from the population and classying each sampling unit into its respective level (subgroup), and we then perform the Pearson $\chi^2$ test.

  • The test involves the following notation:
    • $n$ is the overall sample size

    • $O_j$ is the observed count of each level in the sample

    • $E_j$ is the expected counts under the null hypothesis, and is expressed as $$E_j=N\cdot(p_j)_0,\quad j=1,2,\dotsc,k$$

    • The test statistic is defined as $$\chi^2_{k-1}$$

    • If each level has at least 5 sampling units, then we can assert that $x^2$ (the point estimate for) $\chi^2$ has a $\chi^2$ distribution with $k - 1 $ degrees of freedom.

    • The degrees of freedom is: $k - 1 $.

    The Basics:
  • The following example will perform the Pearson $\chi^2$ test using a flower dataset that has 381 observations of flower phenotype.

  • I will demonstrate how to "manually" calculate the values needed to perform the Pearson $\chi^2$ test in R, and then use the build in functions to perform the same calculations in less steps.

"Manual Calculations" on Cell Frequencies in R¶

  • Test the hypothesis that the sampling proportions is equal to the population proportion
  • The populatio proportion is $p_1 = \frac {9}{16}$, $p_2 = \frac {3}{16}$, $p_3 = \frac {3}{16}$, $p_4 = \frac {1}{16}$
  • Create a vector that contains the count for each level in the sample of 381 observations
  • Create a vector with the populatio proportion
  • use chisq.test(obs, p= prop) to get the association test statistic and p-value
  • The p.value is less than 0.01 and 0.05, so we can reject the $H_0$ in favor of the alternative hypothesis that the sampling proportion is different than the population proportion

Exact Test.


    The Basics:
  • The exact test is used when the sampled data does not have a "large" enough sample size to suppose that the data comes from a $\chi^2$ distribution. (Meaning we dont have 5 sampling units for each categorical level-sub group).

  • In the exact test, the p-values are calcualted directly from the multinomial distribution.

  • This method is calculation intensive, so this test is only recommended when the sample size is small.

  • This method also requires the install.packages('EMT') and that you load the library('EMT') library, which will let us use the multinomial.test function.

  • I will use a hypothetical example of students that have a single undergraduate major with four levels (biology, math, computer science, english) and will test the hypothesis $$\begin{aligned}&H_0\colon p_1=0.41,\ p_2=0.32,\ p_3=0.15,\ p_4=0.12\\\text{against }&H_a\colon\text{some deviation from these proportions exists}\end{aligned}$$
