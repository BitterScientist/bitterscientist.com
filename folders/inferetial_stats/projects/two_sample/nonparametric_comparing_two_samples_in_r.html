Introduction to nonparametric procedures on two population.


    The basics
  • Not all inference tests that compare two samples have populations drawn from normal distributions.

  • In this post I will dive into a few nonparametric procedures that can be used when data is not normally distributed.

  • The sign test (median test) allows us to perform a hypothesis test on a population median, and consider the number of events that fall below and above. We use this test in lieu of the t-test for a single population mean.

  • The sign test can be used when the assumptions of normality fail a paired t-test, since the paired test is a hypothesis test on a single sample (the differences) between the two sample populations.

  • Another more powerful test is the Wilcoxon signed rank test and can be used for making inferences on a single population median, or in a paired/dependent sampling scheme.

  • The final test that will be discussed here is the Mann-Whitney U-test, which is the analogue of the two-sample t-test, although this test finds the difference in the median between two populations that are sampled independently from each other.

  • The Mann-Whitney U-test is also called the Wilcoxon rank sum test and the Wilcoxon-Mann-Whitney test.


Sign Test


    The Basics
  • Use this test if you are interested in deciding if the median of the population is different in a paired sampling scheme is 0 or not.

    • Assumptions:
    • The difference comes from a random sample.

    • The difference is continous


    • Hypothesis test:
    • Null Hypothesis. $H_o: \tilde{\mu_{D}} = 0 $

    • Alternative Hypothesis 2 sided: $H_a: \tilde{\mu_D} \neq 0 $

    • Alternative Hypothesis 1 sided lower tail: $H_a: \tilde{\mu_D} < 0 $

    • Alternative Hypothesis 1 sided upper tail: $H_a: \tilde{\mu_D} > 0 $


  • R method: binom.test(x, n, p=P0, alt=TYPE, conf.level=CL)

Wilcoxon Signed Rank Test


    The Basics
  • Use the Wilcoxon Signed Rank Test on a paired sampling scheme or a single sample, but its similar to a paired t-test for data that is not normally distributed.

  • The Wilcoxon signed rank test is a test for the population median, with the assumption that the differences have a symmetric distribution.
    • Assumptions:
    • The differences comes from a random sample.

    • The differences is either numeric or ordinal

    • The differences is symmetric


    • Hypothesis test:
    • Null Hypothesis. $H_o: \tilde{\mu_{D}} = 0 $

    • Alternative Hypothesis 2 sided: $H_a: \tilde{\mu_D} \neq 0 $

    • Alternative Hypothesis 1 sided lower tail: $H_a: \tilde{\mu_D} < 0 $

    • Alternative Hypothesis 1 sided upper tail: $H_a: \tilde{\mu_D} > 0 $


  • R method: wilcox.test(before, after, paired=TRUE, alt=TYPE, conf.level=CL)

Wilcoxon-Mann-Whitney Rank Sum Test


    The Basics
  • Use this test when you have two independent samples whose data is not normally distributed.

  • The Wilcoxon signed rank test is a test for the population median, with the assumption that the differences have a symmetric distribution.
    • Assumptions:
    • Random sample with same variable from populations

    • The samples are independent form one another.

    • The data is either numeric or ordinal

    • The two populations curves have the same "shape".


    • Hypothesis test:
    • Null Hypothesis. $H_o: \tilde{\mu_1} = \tilde{\mu_2} $

    • Alternative Hypothesis 2 sided: $H_a: \tilde{\mu_1} \neq \tilde{\mu_2} $

    • Alternative Hypothesis 1 sided lower tail: $H_a: \tilde{\mu_1} < \tilde{\mu_2} $

    • Alternative Hypothesis 1 sided upper tail: $H_a: \tilde{\mu_1} > \tilde{\mu_2} $


  • R method: wilcox.test(chol~group, alt='g')
