Introduction to comparing population means with dependent samples.


    The basics
  • This test has two populations with the same numeric variable of interest being observed, but the two observation is for the same sampling unit.

  • The two levels can be thought of as two populations.

  • The categorical variable is the independent variable.

  • The numeric variable we are interested in the comparison, is the response variable, and is commonly called the dependent variable.

  • The big difference with this population mean comparison is that the numeric variables are not independent sampling units, rather, they represent two measurements taken from the same sample
  • Example: Comparing the exam score of a student before and after a workshop series. The sampling unit is the student, and there are two of the same numeric measurement for the same student.

    • Data Organization
    • each instance or row is a sampling unit
    • one column is the response variable from one population (example a time point)
    • second column is the response variable from the second population (example a later time point)
    • Both columns correspond to the same sampling unit
    • the sample mean difference is denoted by
    • the sample mean difference stndard deviation is denoted by $s_d$

  • This test makes inferneces on the difference between the same sampling units population means.

    • Assumptions
    • Data comes for a random sample
    • The two measurements are made one each sampling unit
    • The difference in the two populations is normally distributed

  • The R programming language will be used to carry out these calculations.

    • Hypothesis test:
    • Null Hypothesis. $H_o:\ \mu_D = 0 $

    • Alternative Hypothesis 2 sided: $H_o:\mu_D \neq 0 $

    • Alternative Hypothesis 1 sided lower tail: $H_o:\ \mu_D < 0 $

    • Alternative Hypothesis 1 sided upper tail: $H_o:\ \mu_D > 0 $


    • Hypothesis test values:
    • Parameter of interest. $\mu_D$

    • Point estimate. $\bar{d}$

    • Standard error under $H_0$: $s_d/\sqrt{n}$

    • Test statistic. $$t=\frac{\bar d}{s_d/\sqrt{n}}$$

    • Null distribution: student's t distribution

    • Degrees of freedom: $$df=n-1$$


    • Confidence intervals:

    • Calculated as $100(1- \alpha)$

    • Two-sides: $$\left(\bar d-t_{df,\alpha/2}\widehat{\mathsf{SE}},\bar d+t_{df,\alpha/2}\widehat{\mathsf{SE}}\right)$$

    • Upper one-sided: $$\left(-\infty, \bar d+t_{df,\alpha}\widehat{\mathsf{SE}}\right)$$

    • Lower one-sided: $$\left(\bar d-t_{df,\alpha}\widehat{\mathsf{SE}},\infty\right)$$



R method to compare the population mean with dependent samples.


    The code
  • t.test(before, after, paired=TRUE, alt=TYPE, conf.level=CL)

  • Example: cholesterol measurements are obtained from patients before and after a treatment.

  • Did the treatment reduce the cholesterol.

Calculate parameters, test statistic, and p-values¶

In [3]:
diff <- before-after
dbar <- mean(diff); dbar
s.d <- sd(diff); s.d
n <- length(diff); n
df <- n -1; df
t.crit <- qt(0.05, df, lower.tail=FALSE); t.crit
t.stat <- dbar/(s.d/sqrt(n)); t.stat
p.value <- pt(t.stat, df, lower.tail=FALSE); p.value
