Database Implementation - Import and Export CSV from Sqlite

Import CSV data into DB Browser Sqlite Table

  • Open DB Browser for Sqlite
  • Step 1: Create table: Go to File, then select New In-Memory Database
  • Step 2: Define TableName table, add fields, define fields, click ok
  • Step 3: Import CSV Data
  • Go to File, Select Import, Select Table from CSV File, Select CSV File
  • Step 4: Set Import Settings
  • Add table name, confirm CSV format settings

Export Sqlite Table to CSV

  • Step 1: Confirm Sqlite Table
  • Step 2: Export to CSV
  • File, Export, Table(s) as CSV file...
  • Step 3: Confirm Export Settings
  • Step 4: Confirm CSV Export
